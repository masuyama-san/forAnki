# Card: docker buildでキャッシュを使用せずにビルドするオプションは？

## Question
docker buildでキャッシュを使用せずにビルドするオプションは？

## Answer
docker build --no-cache<br><pre><code>docker build --no-cache -t myapp .  # キャッシュなしでクリーンビルド
docker build --no-cache -t myapp:fresh .  # 最新状態でビルド</code></pre>
