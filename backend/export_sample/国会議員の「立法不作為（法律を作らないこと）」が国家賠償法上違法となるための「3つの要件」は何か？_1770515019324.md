# Card: 国会議員の「立法不作為（法律を作らないこと）」が国家賠償法上違法となるための「3つの要件」は何か？

## Question
国会議員の「立法不作為（法律を作らないこと）」が国家賠償法上違法となるための「3つの要件」は何か？

## Answer
- 権利行使のために立法措置が **必要不可欠** であること。
- その必要性が **明白** であること（不作為の違憲性が明白）。
- 正当な理由なく **長期間** にわたって立法を怠ったこと。

 **趣旨** : 憲法上の権利が、法律の欠如によって行使できない状態（立法による具体化が必要な権利）を解消させるため、厳格な要件下で違法性を認める。
 **解説** : 在外選挙権制限違憲訴訟で定立された規範。単に法律がないというだけでは違法とならない。「不可欠・明白・長期放置」の3要件を満たす場合に限り、国会の裁量権の逸脱として、例外的に国家賠償法上の違法となる。論文式試験における最重要規範の一つ。
 **条文** : 国家賠償法1条1項
 **出典** : 最大判平17.9.14（在外選挙権制限違憲訴訟）
