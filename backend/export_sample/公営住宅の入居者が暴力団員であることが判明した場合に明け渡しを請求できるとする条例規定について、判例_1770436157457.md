# Card: 公営住宅の入居者が暴力団員であることが判明した場合に明け渡しを請求できるとする条例規定について、判例

## Question
公営住宅の入居者が暴力団員であることが判明した場合に明け渡しを請求できるとする条例規定について、判例（西宮市市営住宅事件）の結論と理由は？

## Answer
合憲。
「必要かつ合理的な制限」であるため。

 **趣旨** : 暴力団の活動拠点を排除し、他の住民の平穏な生活を守るという「消極的警察目的」の正当性が認められるため。
 **解説** : 居住の自由に対する制約ではあるが、公営住宅という「特定の場所・公的給付」からの排除にとどまり、市内の他の場所への居住まで禁じるものではない（代替手段がある）点も考慮され、過度な制約ではないとされた。居住の自由を全否定するものではない点がポイント。
 **条文** : 憲法22条1項
