# Card: 民事裁判において、取材源の情報は保護されるか？

## Question
民事裁判において、取材源の情報は保護されるか？

## Answer
民事訴訟法197条1項3号の**「職業の秘密」 **として保護され、証言拒絶が認められる** 場合がある**。

 **趣旨** : 民事事件では、刑事ほど強力な「真実発見の要請」が働かないため、取材の自由との比較衡量の余地が広い。
 **解説** : 刑事（石井記者事件）と結論が異なる重要論点。最高裁は、民事訴訟では取材源の秘密が「職業の秘密」に当たりうるとした上で、①その証言の必要性（民事裁判上の価値）と、②取材源を秘匿する価値（取材の自由）を **比較衡量** し、後者が勝る場合は証言拒絶を認めるとした。
 **条文** : 民事訴訟法197条1項3号
 **出典** : NHK記者証言拒絶事件（最決平18.10.3）
