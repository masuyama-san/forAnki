# Card: 立川反戦ビラ事件（自衛隊官舎への投函）において、住居侵入罪の成立を認めることは合憲か？

## Question
立川反戦ビラ事件（自衛隊官舎への投函）において、住居侵入罪の成立を認めることは合憲か？

## Answer
**合憲** 。
官舎敷地はパブリック・フォーラムではなく、管理者の意思に反する立ち入りは **管理権を侵害する** から。

 **趣旨** : 表現の自由といえども、他人の私生活の平穏や管理権を侵害する「手段」までは保障されない。
 **解説** : 判例は、自衛隊官舎の敷地は「一般に人が自由に出入りできる場所」ではないとした。管理者の意思に反する立ち入りは管理権の侵害であり、表現の内容（政治的意見）に関わらず処罰対象となるとした。
 **条文** : 憲法21条1項、刑法130条
 **出典** : 最判平20.4.11（立川反戦ビラ事件）
