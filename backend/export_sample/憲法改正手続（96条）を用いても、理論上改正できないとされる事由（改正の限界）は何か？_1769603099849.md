# Card: 憲法改正手続（96条）を用いても、理論上改正できないとされる事由（改正の限界）は何か？

## Question
憲法改正手続（96条）を用いても、理論上改正できないとされる事由（改正の限界）は何か？

## Answer
憲法の基本的アイデンティティ（国民主権、基本的人権の尊重、平和主義）。
 **** 
 **趣旨** : 改正権は憲法制定権力から授けられた「制度化された権力」であり、授与の前提である根本原則を覆すことはできない。 
 **条文** : 第96条、第1条、第11条、第9条。 
 **判例・あてはめ** : 明治憲法から日本国憲法への移行において主権が変わったことが「革命」と呼ばれるのは、本来改正限界を超えているからである。
