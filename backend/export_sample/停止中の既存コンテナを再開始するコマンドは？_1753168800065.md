# Card: 停止中の既存コンテナを再開始するコマンドは？

## Question
停止中の既存コンテナを再開始するコマンドは？

## Answer
docker start<br><pre><code>docker start my-container     # my-containerという名前のコンテナを開始

docker start webserver db     # 複数のコンテナを同時開始</code></pre>
