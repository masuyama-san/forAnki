# Card: Dockerfileでホストのファイルやディレクトリをコンテナにコピーする命令は？

## Question
Dockerfileでホストのファイルやディレクトリをコンテナにコピーする命令は？

## Answer
COPY<br><pre><code>COPY app.py /app/                  # app.pyを/appディレクトリにコピー
COPY . /workspace                  # 現在ディレクトリの全てを/workspaceにコピー
COPY config/* /etc/myapp/          # configディレクトリの中身を/etc/myapp/にコピー</code></pre>
