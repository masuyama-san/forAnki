# Card: CMD命令で環境変数を使用する方法は？

## Question
CMD命令で環境変数を使用する方法は？

## Answer
Shell形式または Exec形式でsh -c を使用<br><code>CMD ["sh", "-c", "コマンド"]<br></code>※指定されたコマンドは、/bin/sh -c を介して実行されるためShell機能が使える<br><pre><code>ENV APP_FILE=app.py

# Shell形式（環境変数自動展開）
CMD python $APP_FILE

# Exec形式（sh -c経由で環境変数使用）
CMD ["sh", "-c", "python $APP_FILE"]

# Exec形式（環境変数展開なし）
CMD ["python", "$APP_FILE"]        # これは動作しない</code></pre>
