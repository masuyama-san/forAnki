# Card: フランス人権宣言（1789年）において、「所有権」の法的性質はどのように規定されていたか？

## Question
フランス人権宣言（1789年）において、「所有権」の法的性質はどのように規定されていたか？

## Answer
神聖かつ不可侵の権利 （所有権絶対の原則）

 **趣旨** : 封建的支配や王権による恣意的な収奪から市民の財産を守り、資本主義経済の基盤を確立するため。
 **解説** : 第17条で、所有権を「神聖かつ不可侵」と定義し、正当な補償なき収用を禁じた。この段階では、所有権に対する「公共の福祉による制約」や「社会的責任」という概念はまだ希薄であり、強力な権利として構成されていた。
 **条文** : フランス人権宣言第17条
