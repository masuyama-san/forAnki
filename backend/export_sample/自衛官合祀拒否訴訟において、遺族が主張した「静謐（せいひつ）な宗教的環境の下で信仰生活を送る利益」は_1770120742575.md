# Card: 自衛官合祀拒否訴訟において、遺族が主張した「静謐（せいひつ）な宗教的環境の下で信仰生活を送る利益」は

## Question
自衛官合祀拒否訴訟において、遺族が主張した「静謐（せいひつ）な宗教的環境の下で信仰生活を送る利益」は認められたか？

## Answer
**認められなかった。** 

信教の自由は、他者の宗教的行為に対して「寛容」であることを要請するものであり、他者を排除する権利を含まない。

 **趣旨** : いわゆる「宗教的人格権」の否定。
 **解説** : 殉職した自衛官を護国神社に合祀する行為に対し、キリスト教徒の遺族が精神的苦痛を訴えた事案。最高裁は、強制や干渉がない限り、他者の宗教活動によって不快感を覚えても、それは受忍すべきものであり、法的保護の対象とはならないとした。
 **条文** : 憲法20条
 **出典** : 自衛官合祀拒否訴訟（最大判昭63.6.1）
