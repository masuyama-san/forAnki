# Card: docker pullで特定のプラットフォーム（アーキテクチャ）を指定するオプションは？

## Question
docker pullで特定のプラットフォーム（アーキテクチャ）を指定するオプションは？

## Answer
docker pull --platform プラットフォーム名<br><pre><code>docker pull --platform linux/amd64 nginx    # AMD64アーキテクチャ版
docker pull --platform linux/arm64 ubuntu   # ARM64アーキテクチャ版
docker pull --platform linux/arm/v7 alpine  # ARMv7アーキテクチャ版</code></pre>
