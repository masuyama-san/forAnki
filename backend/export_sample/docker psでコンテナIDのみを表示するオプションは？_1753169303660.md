# Card: docker psでコンテナIDのみを表示するオプションは？

## Question
docker psでコンテナIDのみを表示するオプションは？

## Answer
docker ps -q<br><code>-q </code>: "quiet"（静かに）<br><pre><code>docker ps -q                    # 実行中コンテナのIDのみ表示
docker ps -aq                   # すべてのコンテナIDを表示
docker stop $(docker ps -q)     # 実行中コンテナを全停止</code></pre>
