# Card: 警察官が大学構内に立ち入る際、原則として必要な手続きは？

## Question
警察官が大学構内に立ち入る際、原則として必要な手続きは？

## Answer
**大学の許諾。** 

（緊急時や、裁判官の令状がある場合を除く）

 **趣旨** : 大学の自治（施設管理権）の尊重。
 **解説** : 大学は治外法権ではないため、正当な捜査（令状執行）は拒めないが、学問の自由を守るため、警察活動には大学側の同意や立会いを求める慣行がある。
 **条文** : 憲法23条
