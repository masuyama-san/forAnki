# Card: 集会の主催者ではなく、反対派の妨害行動（敵意ある聴衆）による混乱が予想される場合、公の施設の利用を不

## Question
集会の主催者ではなく、反対派の妨害行動（敵意ある聴衆）による混乱が予想される場合、公の施設の利用を不許可にできるか？

## Answer
原則としてできない。

 **警察の警備等によってもなお混乱を防止できないなどの「特別な事情」がある場合** に限り、例外的に許される。

 **趣旨** : 妨害者の行動を理由に集会を禁止することは、実質的に妨害者に「拒否権（ヘクラーズ・ビトウ）」を与えることになり、憲法の趣旨に反するため。
 **解説** : 警察には、まず妨害を排除して集会を保護する責務がある。したがって、管理者が「混乱のおそれ」だけで不許可にすることは許されず、警察力を投入しても物理的に防げないような極限的な状況に至って初めて、不許可が正当化される。
 **条文** : 憲法21条1項
 **出典** : 最判平8.3.15（上尾市福祉会館事件）
