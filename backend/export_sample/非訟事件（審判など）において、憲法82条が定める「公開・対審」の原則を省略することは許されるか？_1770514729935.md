# Card: 非訟事件（審判など）において、憲法82条が定める「公開・対審」の原則を省略することは許されるか？

## Question
非訟事件（審判など）において、憲法82条が定める「公開・対審」の原則を省略することは許されるか？

## Answer
許される。

（手続保障の縮減が可能。）

 **趣旨** : 非訟事件は、権利の存否を確定する訴訟とは異なり、迅速な処理やプライバシー保護が重視される性質を持つためである。
 **解説** : 前述の通り、非訟事件も32条の「裁判」に含まれるが、その性質上、常に厳格な訴訟手続（公開の法廷での対審・判決）を経る必要はない。事案の性質に応じて、非公開の手続や簡易な審問によって決定することも憲法上許容される。これにより、調停に代わる審判制度なども合憲とされる。
 **条文** : 憲法32条、82条
 **出典** : 最大決昭35.7.6
