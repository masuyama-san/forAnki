# Card: 職業規制の合憲性判定において、規制目的を「消極目的」と「積極目的」に分類し、審査基準を使い分ける判断

## Question
職業規制の合憲性判定において、規制目的を「消極目的」と「積極目的」に分類し、審査基準を使い分ける判断枠組みを何というか？

## Answer
規制目的二分論

 **趣旨** : 規制の性質（目的）に応じて、裁判所が審査すべき深度（厳格か緩やかか）を決定するためのツール。
<div style="display: inline !important;"> **解説** :

-  **消極目的** : 国民の安全確保など警察的目的。司法判断になじむため厳格に審査する。
-  **積極目的** : 社会政策的な弱者保護など。立法裁量が大きいため緩やかに審査する。
 **条文** : 憲法第22条第1項
