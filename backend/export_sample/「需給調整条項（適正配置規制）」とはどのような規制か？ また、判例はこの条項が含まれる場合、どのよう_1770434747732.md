# Card: 「需給調整条項（適正配置規制）」とはどのような規制か？ また、判例はこの条項が含まれる場合、どのよう

## Question
「需給調整条項（適正配置規制）」とはどのような規制か？
また、判例はこの条項が含まれる場合、どのような目的を含んでいると解釈する傾向があるか？

## Answer
**内容** : 需要と供給のバランスを考慮して、参入（開業）を制限する規制。
 **解釈傾向** : 既存業者の経営安定（積極目的）を含むと解釈する傾向。

 **趣旨** : 競争を制限して経営を守るための条項は、実質的に経済政策（積極目的）の性質を帯びる。
 **解説** : 距離制限などの需給調整条項は、表向きは「衛生確保（消極）」であっても、実質的には過当競争防止による「経営保護（積極）」の機能を果たす。判例（公衆浴場事件等）は、この積極目的の側面を重視することで、合憲性を維持する論理構成をとることが多い。
 **条文** : 憲法第22条第1項
