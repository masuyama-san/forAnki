# Card: 財産権の内容や限界を「政令（命令）」のみによって定めることは許されるか？

## Question
財産権の内容や限界を「政令（命令）」のみによって定めることは許されるか？

## Answer
許されない（国会中心立法の原則違反）

 **趣旨** : 国民の権利義務に関わる重要事項は、国民代表機関である国会の議決を経た「法律」で定めるべきという民主主義の要請（41条）である。
 **解説** : 財産権の制限は国民の権利への重大な介入であるため、原則として法律（または法律の具体的委任を受けた命令）による必要がある。法律の根拠を持たず、内閣等の行政機関が制定する命令（政令・省令）のみによって財産権を侵害・制約することは、国会中心立法の原則に反し違憲となる。
 **条文** : 憲法29条2項、41条
