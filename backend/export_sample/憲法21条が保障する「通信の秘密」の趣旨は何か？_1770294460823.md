# Card: 憲法21条が保障する「通信の秘密」の趣旨は何か？

## Question
憲法21条が保障する「通信の秘密」の趣旨は何か？

## Answer
- **私生活の自由（プライバシー）**の保護。
- 公権力による通信内容の探索の可能性を断つことによる、 **自由な表現伝達手段** の確保。

 **趣旨** : 通信は個人の結合を助ける重要な手段であり、監視の不安があれば表現行為自体が萎縮してしまうため。
 **解説** : 単に「プライバシーを守るため」だけでなく、「表現の自由を実質化するためのインフラ保護」という側面（2点目）を指摘できるかどうかが、論文試験での加点ポイントとなる。
 **条文** : 憲法21条2項後段
