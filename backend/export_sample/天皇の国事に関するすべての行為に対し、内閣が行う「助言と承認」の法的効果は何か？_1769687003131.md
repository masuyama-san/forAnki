# Card: 天皇の国事に関するすべての行為に対し、内閣が行う「助言と承認」の法的効果は何か？

## Question
天皇の国事に関するすべての行為に対し、内閣が行う「助言と承認」の法的効果は何か？

## Answer
- 天皇に対する拘束力（天皇は拒否できない）
- 政治的責任の帰属（内閣が一切の責任を負う）

 **趣旨** : 天皇から政治的実権を剥奪し、内閣が国政の全責任を負うことで、民主主義と天皇の非政治性を担保するため。 
 **解説** : 憲法3条は「内閣の助言と承認を必要とし、内閣が、その責任を負ふ」と定める。これにより、天皇は内閣の意思に拘束され、自らの意思で行為を拒むことは認められない（拒否権の否定）。万が一、天皇が内閣の助言に反する行為をした場合でも、その法的・政治的責任はすべて内閣に帰属する。試験対策上は、この責任帰属が「天皇の無答責」の裏返しであることを押さえる必要がある。 
 **条文** : 憲法3条
