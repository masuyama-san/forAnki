# Card: ワイマール憲法（1919年）において、所有権の性質はフランス人権宣言（神聖不可侵）からどう修正された

## Question
ワイマール憲法（1919年）において、所有権の性質はフランス人権宣言（神聖不可侵）からどう修正されたか？

## Answer
所有権は義務を伴う（社会拘束性）。 
「その行使は、同時に公共の福祉に役立つべきものである」

 **趣旨** : 私有財産権を認めつつも、その行使が社会全体の利益と調和することを義務付けるため。
 **解説** : 所有権絶対の原則を修正し、財産権の内容は法律で定められ、公共の福祉による制約を受けることを明記した。これは現代の日本国憲法第29条（財産権）の規定にも直接的な影響を与えている。
 **条文** : ワイマール憲法第153条
