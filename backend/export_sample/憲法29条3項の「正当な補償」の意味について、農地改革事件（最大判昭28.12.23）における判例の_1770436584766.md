# Card: 憲法29条3項の「正当な補償」の意味について、農地改革事件（最大判昭28.12.23）における判例の

## Question
憲法29条3項の「正当な補償」の意味について、農地改革事件（最大判昭28.12.23）における判例の立場は何か？

## Answer
相当補償説（必ずしも市場価格と一致しなくてよい）

 **趣旨** : 社会全体の構造を変革するような特殊な政策下においては、国家財政や政策目的との調和を図る必要があるためである。
 **解説** : 農地改革という戦後の抜本的な社会変革事業においては、合理的に算出された「相当な額」であれば、当時の市場価格（自由取引価格）と一致しなくても「正当な補償」にあたるとされた。※ただし、土地収用のような平時の個別的徴収においては、完全補償（市場価格）が必要というのが通説・判例の傾向である。
 **条文** : 憲法29条3項
