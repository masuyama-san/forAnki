# Card: 土地収用法など、通常の公用収用において憲法29条3項が要求する「正当な補償」の内容は何か？

## Question
土地収用法など、通常の公用収用において憲法29条3項が要求する「正当な補償」の内容は何か？

## Answer
完全補償（市場価格による全額補償）

 **趣旨** : 特定の個人の財産を公共のために奪う以上、その損失を完全に填補しなければ、実質的な私有財産制の保障とならないからである。
 **解説** : 農地改革のような例外的な場合を除き、平時の公用収用（道路建設のための土地収用など）においては、収用される財産の客観的な市場価格に加え、移転料などの付随的損失も含めた「完全な補償」が必要であると解されている（通説・下級審判例等）。
 **条文** : 憲法29条3項
