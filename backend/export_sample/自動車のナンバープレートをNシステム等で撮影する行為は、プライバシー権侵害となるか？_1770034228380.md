# Card: 自動車のナンバープレートをNシステム等で撮影する行為は、プライバシー権侵害となるか？

## Question
自動車のナンバープレートをNシステム等で撮影する行為は、プライバシー権侵害となるか？

## Answer
原則として **侵害とならない** 。

 **趣旨** : ナンバープレートは外部から視認・識別されることを目的とした情報だから。
 **解説** : 自動車ナンバーは、公道を走行する以上、他者に知られることが予定されている外形的事実であり、個人の内面に関わる秘密情報ではない。したがって、これを撮影しても直ちにプライバシー権侵害とはならない。
 **条文** : 憲法13条
