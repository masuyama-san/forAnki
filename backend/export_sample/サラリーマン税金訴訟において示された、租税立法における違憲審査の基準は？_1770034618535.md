# Card: サラリーマン税金訴訟において示された、租税立法における違憲審査の基準は？

## Question
サラリーマン税金訴訟において示された、租税立法における違憲審査の基準は？

## Answer
立法目的が正当であり、区別が **著しく不合理であることが明らか** でない限り合憲とする。 （明白性の原則）

 **趣旨** : 租税制度の構築には、国の財政事情や高度に専門技術的な判断が必要であり、立法府（国会）の広範な裁量を尊重すべきだから。
 **解説** : 租税法の分野では、裁判所は立法府の裁量を広く認め、介入を控える。本件では、給与所得者と事業所得者の間の必要経費控除の仕組みの違い（クロヨン問題等）について、著しく不合理とはいえないとして合憲とした。
 **条文** : 憲法14条1項、84条
 **出典** : 最大判昭60.3.27
