# Card: Dockerイメージを削除するコマンドは？

## Question
Dockerイメージを削除するコマンドは？

## Answer
docker rmi<br><pre><code>docker rmi nginx               # nginxイメージを削除
docker rmi my-app:v1.0         # 特定バージョンのイメージを削除
docker rmi ubuntu:20.04        # Ubuntu 20.04イメージを削除</code></pre>
