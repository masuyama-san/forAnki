# Card: CMD命令をDockerfileで複数記述した場合の動作は？

## Question
CMD命令をDockerfileで複数記述した場合の動作は？

## Answer
最後のCMD命令のみが有効になる<br><pre><code>CMD ["echo", "first"]              # この命令は無効
CMD ["echo", "second"]             # この命令は無効
CMD ["echo", "third"]              # この命令のみが実行される

# 実行結果：third のみが出力される</code></pre>
