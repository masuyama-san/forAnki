# Card: docker rmで実行中のコンテナも強制削除するオプションは？

## Question
docker rmで実行中のコンテナも強制削除するオプションは？

## Answer
docker rm -f<br><code>-f </code>: "force"（強制）<br><pre><code>docker rm -f running-container  # 実行中でも強制削除
docker rm --force webapp        # 強制削除（実行中でもOK）
docker rm -f $(docker ps -aq)   # すべてのコンテナを強制削除</code></pre>
