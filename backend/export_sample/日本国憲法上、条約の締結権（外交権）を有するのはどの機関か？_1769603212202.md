# Card: 日本国憲法上、条約の締結権（外交権）を有するのはどの機関か？

## Question
日本国憲法上、条約の締結権（外交権）を有するのはどの機関か？

## Answer
内閣
 **** 
 **趣旨** : 行政権の属する内閣が、国家を代表して迅速かつ機動的に外交を行う。 
 **条文** : 第73条3号。 
 **判例・あてはめ** : ただし、民主的コントロールのため、原則として「国会の事前承認（時宜によっては事後承認）」が必要である。
