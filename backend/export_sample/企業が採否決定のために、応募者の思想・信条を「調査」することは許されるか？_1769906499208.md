# Card: 企業が採否決定のために、応募者の思想・信条を「調査」することは許されるか？

## Question
企業が採否決定のために、応募者の思想・信条を「調査」することは許されるか？

## Answer
**許される** 。 
採用の自由には、そのための **調査の自由** も含まれるから。

 **趣旨** : 契約締結の判断に必要な資料を収集する活動も、経済活動の自由として保護するため。
 **解説** : 企業が誰を雇うかを決める自由を持つ以上、その判断のために思想・信条を調査することも原則として自由である。調査自体が直ちに公序良俗違反となるわけではないとした。
 **条文** : 民法90条
 **出典** : 最大判昭48.12.12（三菱樹脂事件）
