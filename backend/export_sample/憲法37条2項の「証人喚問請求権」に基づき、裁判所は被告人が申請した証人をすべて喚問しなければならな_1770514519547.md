# Card: 憲法37条2項の「証人喚問請求権」に基づき、裁判所は被告人が申請した証人をすべて喚問しなければならな

## Question
憲法37条2項の「証人喚問請求権」に基づき、裁判所は被告人が申請した証人をすべて喚問しなければならないか？

## Answer
**すべて喚問する必要はない** 。

（裁判所の合理的な証拠採否の裁量が認められる）

 **趣旨** : 迅速かつ適正な裁判（37条1項）を実現するため、無益・不要な証人まで取り調べる義務を裁判所に課すものではない。
 **解説** : 37条2項は、被告人の防御に「必要かつ適切な」証人を喚問する権利を保障するものです。不必要な証人を却下することは裁判所の訴訟指揮権の範囲内であり、違憲とはなりません。
 **条文** : 憲法37条2項
 **出典** : 最大判昭23.7.29
