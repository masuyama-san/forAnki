# Card: 堀木訴訟において、社会保障給付（併給禁止）の合憲性を判断する際に示された基準は？

## Question
堀木訴訟において、社会保障給付（併給禁止）の合憲性を判断する際に示された基準は？

## Answer
立法府の広い裁量に委ねられており、 **著しく合理性を欠き、裁量の逸脱・濫用が明白な場合** を除き合憲。

 **趣旨** : 限られた財源をどう配分するかは、高度に政治的な判断（立法裁量）に委ねられるため。
 **解説** : 憲法25条（生存権）の具体化にあたっては、立法府に広い裁量が認められる。児童扶養手当と障害福祉年金の併給を禁止することは、偶発的な事情による二重給付を避ける等の政策的意図に基づくものであり、立法裁量の範囲内として **合憲** と判断された。
 **条文** : 憲法14条1項、25条
 **出典** : 最大判昭57.7.7
