# Card: 皇室が財産を譲り受け、または賜与する際、憲法8条により必要とされる手続は何か？

## Question
皇室が財産を譲り受け、または賜与する際、憲法8条により必要とされる手続は何か？

## Answer
国会の議決

 **** 
 **趣旨** : 皇室と特定の政治勢力の結びつきを防止し、皇室経済を民主的統制の下に置くため。 
 **解説** : 憲法8条は、皇室の財産授受について「国会の議決」を要求している。これは88条が皇室費用の「予算」としての包括的議決を定めているのに対し、個別の贈与や受領を監視するための規定である。司法試験の短答式では、8条が「通常の議決」を指すのに対し、88条は「予算としての議決」であるという、審議手続（衆議院の先議権の有無など）の違いが問われやすい。 
 **条文** : 憲法8条、88条
