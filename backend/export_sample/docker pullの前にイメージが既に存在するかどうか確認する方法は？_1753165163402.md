# Card: docker pullの前にイメージが既に存在するかどうか確認する方法は？

## Question
docker pullの前にイメージが既に存在するかどうか確認する方法は？

## Answer
docker images または docker image ls<br><pre><code>docker images nginx            # nginxイメージの確認
docker pull nginx              # 最新版があればダウンロード
docker images                  # ダウンロード後の確認</code></pre>
