# Card: 憲法前文の法的性質について、「法規範性」と「裁判規範性」は認められるか？

## Question
憲法前文の法的性質について、「法規範性」と「裁判規範性」は認められるか？

## Answer
- 法規範性：認める（憲法の一部をなすため）。
- 裁判規範性：原則として認めない（抽象的な宣言に留まるため）。

 **趣旨** : 憲法の基本原理を宣言する前文の効力を認めつつ、具体的訴訟での濫用を防ぐため。 
 **解説** : 前文は憲法の一部であり、法的な効力（法規範性）を有することは一般に認められる。しかし、裁判所が具体的事件を解決する際の基準として直接適用できる力（裁判規範性）については、判例（長沼ナイキ事件等）は慎重である。前文の平和的生存権などを根拠に直接訴えを提起しても、具体的権利性が否定される傾向にある。ただし、個別条文（9条等）の解釈指針としての役割は否定されていない。 
 **条文** : 憲法前文
