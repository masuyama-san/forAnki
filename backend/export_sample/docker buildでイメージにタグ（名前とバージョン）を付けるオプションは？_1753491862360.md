# Card: docker buildでイメージにタグ（名前とバージョン）を付けるオプションは？

## Question
docker buildでイメージにタグ（名前とバージョン）を付けるオプションは？

## Answer
docker build -t タグ名<br><code>-t (--tag)</code>: "tag"（タグ）<br><pre><code>docker build -t myapp:1.0 .         # バージョン1.0でタグ付け
docker build -t mycompany/webapp:latest .  # 会社名付きでタグ付け
docker build -t myapp:prod -t myapp:latest .  # 複数タグを同時に付与</code></pre>
