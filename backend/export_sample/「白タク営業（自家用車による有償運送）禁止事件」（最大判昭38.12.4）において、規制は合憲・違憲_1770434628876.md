# Card: 「白タク営業（自家用車による有償運送）禁止事件」（最大判昭38.12.4）において、規制は合憲・違憲

## Question
「白タク営業（自家用車による有償運送）禁止事件」（最大判昭38.12.4）において、規制は合憲・違憲のどちらとされたか？またその主たる目的は？

## Answer
-  **結論** : 合憲
-  **目的** : 消極目的（道路運送の秩序維持と利用者の安全確保）

 **趣旨** : 免許のない事業者が運送を行うことは、国民の生命・身体に直接的な危険を及ぼすため。
 **解説** : 白タク営業の禁止は、運行管理や整備能力のない者による運送を防ぎ、公衆の安全（消極目的）を守るために必要かつ合理的な措置であるとして合憲とされた。また、道路運送の総合的な発達（積極的側面）も考慮されている。
 **条文** : 憲法第22条第1項
 **出典** : 最大判昭38.12.4
