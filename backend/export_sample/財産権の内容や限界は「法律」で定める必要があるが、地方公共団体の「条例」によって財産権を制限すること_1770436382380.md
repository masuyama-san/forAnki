# Card: 財産権の内容や限界は「法律」で定める必要があるが、地方公共団体の「条例」によって財産権を制限すること

## Question
財産権の内容や限界は「法律」で定める必要があるが、地方公共団体の「条例」によって財産権を制限することは許されるか？

## Answer
許される

 **趣旨** : 地域の特殊性に応じたきめ細かい規制を行う必要性があり、条例も民主的基盤（住民代表性）を有する法形式だからである。
 **解説** : 憲法29条2項は「法律」と規定しているが、判例は、地方公共団体の事務として処理することが必要かつ適当な場合には、法律の授権がなくとも、条例によって財産権を制限（ため池の堤とうの使用禁止等）することを認めている。ただし、国の法令に違反しない範囲に限られる。これは地方自治の本旨（憲法94条）とも調和する解釈である。
 **条文** : 憲法29条2項、憲法94条
