# Card: 国会議員の「立法行為（法律の制定）」が、国家賠償法1条1項の「違法」となるのは、どのような場合か？

## Question
国会議員の「立法行為（法律の制定）」が、国家賠償法1条1項の「違法」となるのは、どのような場合か？

## Answer
原則：違法とならない。

例外：容易に想定し難いような例外的な場合。

（違憲性が明白であるにもかかわらず、あえて立法を行った場合など。）

 **趣旨** : 国会議員が事後的な賠償責任を恐れて萎縮することなく、自由に国民のための立法活動を行えるようにするためである。
 **解説** : 法律の内容が憲法違反であっても、直ちにその立法行為が国家賠償法上の違法となるわけではない。国会議員の職務上の義務違反（違法）が認められるのは、立法の内容が憲法の一義的な文言に違反しているにもかかわらずあえて立法を行った場合など、極めて例外的な場合に限られる。
 **条文** : 国家賠償法1条1項
 **出典** : 最判昭60.11.21（在宅投票制度廃止事件）
