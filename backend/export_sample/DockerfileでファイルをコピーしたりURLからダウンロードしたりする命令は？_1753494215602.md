# Card: DockerfileでファイルをコピーしたりURLからダウンロードしたりする命令は？

## Question
DockerfileでファイルをコピーしたりURLからダウンロードしたりする命令は？

## Answer
ADD<br><pre><code>ADD app.tar.gz /app/               # tar.gzを自動展開してコピー
ADD http://example.com/file.txt /app/  # URLからファイルをダウンロード
ADD local-file.txt /app/           # ローカルファイルをコピー</code></pre>
