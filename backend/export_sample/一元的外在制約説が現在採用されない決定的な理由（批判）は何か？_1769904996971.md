# Card: 一元的外在制約説が現在採用されない決定的な理由（批判）は何か？

## Question
一元的外在制約説が現在採用されない決定的な理由（批判）は何か？

## Answer
明治憲法下の「法律の留保」と同様の結果を招き、人権保障が形骸化するから。

 **趣旨** : 立法権であっても侵せない「永久の権利」としての憲法保障の実質を守るため。
 **解説** : 外在説では「公共の福祉（公益）」の内容を決めるのは国会（法律）となるため、「法律さえ作れば公益の名の下に人権を制限できる」という結論になりがちである。これでは「法律の範囲内でのみ人権を認める」とした明治憲法と実質的に変わらなくなってしまう。
 **条文** : 憲法11条、97条
