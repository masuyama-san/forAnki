# Card: 「許可制」は、職業選択の自由そのものに対する強力な制限であるため、判例上、原則としてどのような要件下

## Question
「許可制」は、職業選択の自由そのものに対する強力な制限であるため、判例上、原則としてどのような要件下でのみ許容されるか？

## Answer
重要な公共の利益のために必要かつ合理的であること

 **趣旨** : 許可制は「原則禁止・例外解除」のシステムであり、自由を奪う強度が極めて高いため。
 **解説** : 職業活動の方法の規制ではなく、職業に就くこと自体を制約する「許可制」は、最も強力な規制手段の一つである。したがって、その導入には単なる便宜ではなく、「重要な公共の利益」という高いハードルが必要となる。
 **条文** : 憲法第22条第1項
 **出典** : 最大判昭50.4.30
