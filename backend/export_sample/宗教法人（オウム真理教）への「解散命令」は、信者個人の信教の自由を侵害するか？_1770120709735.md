# Card: 宗教法人（オウム真理教）への「解散命令」は、信者個人の信教の自由を侵害するか？

## Question
宗教法人（オウム真理教）への「解散命令」は、信者個人の信教の自由を侵害するか？

## Answer
**侵害しない（合憲）。** 

解散命令は法人の「世俗的側面（法人格）」を失わせるにとどまり、信者個人の信仰や宗教行為（精神的側面）そのものを禁止するものではないから。

 **趣旨** : 宗教団体の二重性（宗教的側面／世俗的法人側面）の区別。
 **解説** : 解散命令によって宗教活動に事実上の支障が生じることは認めたが、それは「間接的・事実上の効果」にすぎないとした。法令違反行為に基づく解散命令は、公共の福祉による必要かつ合理的な規制であり、合憲である。
 **条文** : 憲法20条1項
 **出典** : オウム真理教解散命令事件（最決平8.1.30）
