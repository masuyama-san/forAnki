# Card: ENV命令の2つの記述形式は？

## Question
ENV命令の2つの記述形式は？

## Answer
ENV key=value（推奨）
<br>ENV key value（スペース区切り）<br><pre><code># key=value形式（推奨）
ENV NODE_ENV=production
ENV APP_PORT=8080
ENV DEBUG=false

# key value形式
ENV NODE_ENV production
ENV APP_PORT 8080</code></pre>
