# Card: 天皇が国会を「召集」する際、その決定権（実質的決定権）はどこにあるか？

## Question
天皇が国会を「召集」する際、その決定権（実質的決定権）はどこにあるか？

## Answer
内閣

 **趣旨** : 国会の活動開始時期を政府が責任を持って決定し、天皇を政治判断から切り離すため。 
 **解説** : 憲法7条2号は天皇の国事行為として「国会を召集すること」を定める。天皇は形式的な召集詔書を発するのみであり、いつ、どのような国会を召集するかという実質的な決定は内閣が行う（憲法53条参照）。このため、内閣の決定（助言と承認）なしに天皇が自ら国会を召集することは憲法上許されない。
 **条文** : 憲法7条2号、3条、53条
