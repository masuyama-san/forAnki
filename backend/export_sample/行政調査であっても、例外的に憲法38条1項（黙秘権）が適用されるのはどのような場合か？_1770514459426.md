# Card: 行政調査であっても、例外的に憲法38条1項（黙秘権）が適用されるのはどのような場合か？

## Question
行政調査であっても、例外的に憲法38条1項（黙秘権）が適用されるのはどのような場合か？

## Answer
実質上、 **刑事責任追及のための資料収集** に直接結びつく作用を一般的に有する手続である場合。

 **趣旨** : 名目が「行政調査」であっても、実態が「犯則調査（犯罪捜査）」と同視できる場合には、脱法的な人権侵害を防ぐため黙秘権を保障する必要がある。
 **解説** : 判例は、手続の形式的名称ではなく「実質的な機能」で判断します。調査の主眼が「犯罪の証拠収集」にあると認定されれば、38条が適用され、黙秘権の告知なしに行われた供述の証拠能力などが否定される可能性があります。
 **条文** : 憲法38条1項
 **出典** : 最大判昭47.11.22（川崎民商事件）
