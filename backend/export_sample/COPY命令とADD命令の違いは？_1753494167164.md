# Card: COPY命令とADD命令の違いは？

## Question
COPY命令とADD命令の違いは？

## Answer
COPY：シンプルなファイルコピーのみ
<br>ADD：URL取得、tar自動展開などの追加機能あり<br>（ほとんどの場合、ADD よりも COPY の使用が推奨されます。）<br><pre><code># COPY（推奨：シンプル）
COPY app.tar.gz /app/              # ファイルをそのままコピー

# ADD（特殊機能が必要な場合のみ）
ADD app.tar.gz /app/               # tar.gzを自動展開してコピー
ADD http://example.com/file.txt /app/  # URLからダウンロード</code></pre>
