# Card: 労働組合 が、特定候補者の支援のために「選挙資金」を組合員から強制徴収することは許されるか？

## Question
**労働組合** が、特定候補者の支援のために**「選挙資金」**を組合員から強制徴収することは許されるか？

## Answer
許されない（違法）。

 **趣旨** : 「金銭的寄付」の強制は、組合員個人の「思想・信条の自由（政治的資金拠出の自由）」を直接的に侵害するため。
 **解説** : 組合が政治活動を行うこと（前問）は自由だが、その費用の負担まで強制することは、反対派組合員に「意に反する政治的寄付」を強いることになり、協力義務の限界を超える。活動はOKだが、カネの強制はNGという区分けが重要。
 **条文** : 憲法19条
