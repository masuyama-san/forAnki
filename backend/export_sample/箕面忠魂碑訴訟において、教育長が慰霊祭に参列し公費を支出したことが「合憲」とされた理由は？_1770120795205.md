# Card: 箕面忠魂碑訴訟において、教育長が慰霊祭に参列し公費を支出したことが「合憲」とされた理由は？

## Question
箕面忠魂碑訴訟において、教育長が慰霊祭に参列し公費を支出したことが「合憲」とされた理由は？

## Answer
当該行為が、宗教的活動ではなく**「社会的儀礼」**の範囲内であると判断されたため。

 **趣旨** : 公職者の行為のすべてが「宗教的活動」になるわけではない。
 **解説** : 遺族に対する応接や慰謝の措置として行われたものであり、特定の宗教を援助・助長する意図も効果もないとされた。「社会的儀礼」というキーワードが出たら、合憲（政教分離違反ではない）の方向で論証する。
 **条文** : 憲法20条3項
 **出典** : 箕面忠魂碑訴訟（最判平5.2.16）
