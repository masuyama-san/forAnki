# Card: 政教分離原則（20条・89条）における「目的・効果基準」とは？

## Question
政教分離原則（20条・89条）における「目的・効果基準」とは？

## Answer
主に **宗教的儀式・行事** （地鎮祭など）の合憲性判定に用いられる基準。

- 行為の **目的** が宗教的意義を持つか。
- その **効果** が特定の宗教を援助・助長、または圧迫・干渉するものか。

 **趣旨** : 国家と宗教の完全な分離は不可能であるため、社会的・文化的諸条件に照らし「相当とされる限度」を超える関わりのみを排除するため。
 **解説** : 津地鎮祭事件で定立された基準。地鎮祭は「建築着工の際の慣習化した社会的儀礼」であり、目的は世俗的（工事の安全）、効果も特定宗教の援助にはあたらないとして合憲とされた。一方、公有地の提供などの事案では、より厳格な「総合衡量」が用いられる。
 **条文** : 憲法20条3項
 **出典** : 津地鎮祭事件（最大判昭52.7.13）
