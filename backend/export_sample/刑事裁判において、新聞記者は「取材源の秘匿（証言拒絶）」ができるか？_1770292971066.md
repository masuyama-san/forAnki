# Card: 刑事裁判において、新聞記者は「取材源の秘匿（証言拒絶）」ができるか？

## Question
刑事裁判において、新聞記者は「取材源の秘匿（証言拒絶）」ができるか？

## Answer
原則として **認められない** 。

（刑事訴訟法に明文規定がない限り、証言義務が優先する）

 **趣旨** : 刑事裁判における真実発見と、国民の証言義務の公平性は、取材の自由よりも優先される重大な公益であるため。
 **解説** : 石井記者事件において最高裁は、記者の証言拒絶権は憲法21条から直接保障されるものではないとした。刑事訴訟法149条は医師・弁護士等の証言拒絶権を定めているが、記者は含まれない。立法政策の問題であり、現行法下では証言義務を負う。
 **条文** : 憲法21条、刑事訴訟法149条
 **出典** : 石井記者事件（最大判昭27.8.6）
