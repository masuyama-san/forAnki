# Card: 公務員定年制などの「年齢」による区別は、「社会的身分」による差別に当たるか？

## Question
公務員定年制などの「年齢」による区別は、「社会的身分」による差別に当たるか？

## Answer
**当たらない**  （年齢は社会的身分ではないため、緩やかな審査基準で合憲とされる）

 **趣旨** : 年齢に基づく合理的な制度設計（定年制等）を維持するため。
 **解説** : 判例は、高齢であることは「社会的身分（継続的な地位）」には当たらないとする。したがって、年齢による区別は「疑わしい区分」としての厳格な審査ではなく、一般的な**合理性審査（著しく不合理でない限り合憲）**によって処理される。
 **条文** : 憲法14条1項
 **出典** : 最大判昭39.5.27（公務員定年制事件）
