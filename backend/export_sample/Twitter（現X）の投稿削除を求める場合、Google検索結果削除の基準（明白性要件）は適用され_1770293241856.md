# Card: Twitter（現X）の投稿削除を求める場合、Google検索結果削除の基準（明白性要件）は適用され

## Question
Twitter（現X）の投稿削除を求める場合、Google検索結果削除の基準（明白性要件）は適用されるか？

## Answer
適用されない。

プライバシー等の利益が、投稿を閲覧に供し続ける理由よりも **優越** すれば足りる（「明らかに」は不要）。

 **趣旨** : SNSの個別投稿は、検索エンジンのような「社会全体の情報流通インフラ」としての公共性は低いため。
 **解説** : 令和4年判決。Google事件との対比が問われる最重要論点。TwitterのようなSNSについては、検索事業者特有の事情（網羅的収集の必要性）がないため、通常の不法行為や差止めと同様、単なる比較衡量（どちらの利益が重いか）で判断すべきとした。
 **条文** : 憲法21条、13条
 **出典** : Twitter記事削除請求事件（最判令4.6.24）
