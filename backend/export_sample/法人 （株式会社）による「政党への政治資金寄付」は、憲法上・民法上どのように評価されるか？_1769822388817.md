# Card: 法人 （株式会社）による「政党への政治資金寄付」は、憲法上・民法上どのように評価されるか？

## Question
**法人** （株式会社）による「政党への政治資金寄付」は、憲法上・民法上どのように評価されるか？

## Answer
-  **憲法上** ：政治活動の自由の一環として保障される。
-  **民法上** ：定款の「目的の範囲内」の行為として有効。

 **趣旨** : 会社は現代社会の重要な構成要素であり、国政に関心を持ち意見を表明する自由（政治活動の自由）を有するため。
 **解説** : 判例は、会社の「定款の目的」を客観的・抽象的に広く解釈し、政治献金も会社の社会的役割を果たす行為として目的の範囲内に含む。また、会社による献金は株主の政治的自由を直接侵害するものではないとした。
 **条文** : 憲法21条、民法34条
