# Card: 教科書検定（不合格になると教科書として発行不可）は、「検閲」に当たるか？

## Question
教科書検定（不合格になると教科書として発行不可）は、「検閲」に当たるか？

## Answer
**当たらない** 。

- 不合格でも**「一般図書」として出版可能**であり、発表そのものの禁止ではない。
- 思想調査を目的としていない。

 **趣旨** : 教育水準の維持という行政目的のための制度であり、思想統制そのものを目的としないため。
 **解説** : 判例は、「発表禁止目的」や「発表前の審査」といった検閲の特質を欠くとして検閲該当性を否定した。その上で、教育の中立・公正等の要請から21条1項にも反せず合憲とした。
 **条文** : 憲法21条2項、同1項
 **出典** : 最判平5.3.16（教科書裁判第一次訴訟上告審）
