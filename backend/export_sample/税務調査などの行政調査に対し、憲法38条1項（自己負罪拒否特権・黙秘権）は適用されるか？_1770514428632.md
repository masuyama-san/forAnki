# Card: 税務調査などの行政調査に対し、憲法38条1項（自己負罪拒否特権・黙秘権）は適用されるか？

## Question
税務調査などの行政調査に対し、憲法38条1項（自己負罪拒否特権・黙秘権）は適用されるか？

## Answer
**原則として適用されない** 。

 **趣旨** : 38条1項は「刑事手続」における人権規定であり、純粋な行政目的（公平な課税など）の調査とは性質・目的が異なるため。
 **解説** : 川崎民商事件において、所得税法上の質問検査権は「刑事責任追及を目的としない」ため、38条1項は及ばず、検査拒否への罰則も合憲とされました。
 **条文** : 憲法38条1項
 **出典** : 最大判昭47.11.22（川崎民商事件）
