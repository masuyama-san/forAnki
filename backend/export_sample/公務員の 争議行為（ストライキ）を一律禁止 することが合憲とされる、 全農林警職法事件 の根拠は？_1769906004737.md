# Card: 公務員の 争議行為（ストライキ）を一律禁止 することが合憲とされる、 全農林警職法事件 の根拠は？

## Question
公務員の **争議行為（ストライキ）を一律禁止** することが合憲とされる、 **全農林警職法事件** の根拠は？

## Answer
- 地位の特殊性・職務の公共性（ **全体の奉仕者** ）
-  **財政民主主義** （勤務条件は議会が予算で決める）
- 市場抑制力の欠如（倒産がない）
-  **代償措置** の存在（人事院勧告）

 **趣旨** : 公務員の労働基本権（憲法28条）制約を正当化する論理の構成。
 **解説** : 政治活動（表現の自由）については堀越事件で緩和傾向にあるが、労働基本権については全農林判決の「一律禁止合憲」の枠組みが維持されている。特に「給与は税金であり議会が決める（財政民主主義）」という点が、民間労働者と区別する最大の論拠となる。
 **条文** : 国家公務員法98条、憲法28条
