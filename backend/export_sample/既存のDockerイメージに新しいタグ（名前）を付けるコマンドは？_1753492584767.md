# Card: 既存のDockerイメージに新しいタグ（名前）を付けるコマンドは？

## Question
既存のDockerイメージに新しいタグ（名前）を付けるコマンドは？

## Answer
docker tag<br><pre><code>docker tag nginx my-nginx          # nginxにmy-nginxというタグを追加
docker tag my-app:latest my-app:v1.0  # latestタグをv1.0としてもタグ付け
docker tag ubuntu:20.04 my-ubuntu  # Ubuntu 20.04にカスタムタグを付与</code></pre>
