# Card: 八月革命説において、主権の移転（法的革命）が発生した時点はいつか？

## Question
八月革命説において、主権の移転（法的革命）が発生した時点はいつか？

## Answer
ポツダム宣言受諾時（1945年8月）

 **趣旨** : 国際約束である宣言の受諾により、日本国家の根本原則（主権の所在）が国民主権へと切り替わったと捉える。 
 **条文** : 明治憲法第73条（形式的な改正手続として使用）、ポツダム宣言。 
 **判例・あてはめ** : 通説的見解。受諾の瞬間に明治憲法は「根本建前」において死滅し、国民が新たな憲法制定権力を獲得したと説明される。
