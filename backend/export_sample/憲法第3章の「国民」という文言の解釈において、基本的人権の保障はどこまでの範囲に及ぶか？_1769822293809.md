# Card: 憲法第3章の「国民」という文言の解釈において、基本的人権の保障はどこまでの範囲に及ぶか？

## Question
憲法第3章の「国民」という文言の解釈において、基本的人権の保障はどこまでの範囲に及ぶか？

## Answer
権利の性質上、日本国民のみを対象としていると解されるものを除き、外国人や法人にも人権保障が及ぶ。

 **趣旨** : 人権の前国家的性格（天賦人権説）に基づき、条文の文言（「国民」）に拘泥せず実質的に保障範囲を判断するため。
 **解説** : 最高裁はマクリーン事件において、基本的人権の保障は「権利の性質上日本国民のみを対象としていると解されるものを除き」在留外国人にも等しく及ぶとした。法人についても同様に、性質上可能な限り適用される（八幡製鉄事件）。
 **条文** : 憲法第3章総論
