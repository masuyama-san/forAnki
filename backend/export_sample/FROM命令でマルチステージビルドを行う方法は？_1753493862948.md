# Card: FROM命令でマルチステージビルドを行う方法は？

## Question
FROM命令でマルチステージビルドを行う方法は？

## Answer
FROM イメージ名 AS ステージ名<br><pre><code># ビルドステージ
FROM node:16 AS builder
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
RUN npm run build

# 本番ステージ
FROM nginx:alpine AS production
COPY --from=builder /app/dist /usr/share/nginx/html</code></pre>
