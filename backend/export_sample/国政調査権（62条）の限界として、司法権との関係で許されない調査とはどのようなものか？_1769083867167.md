# Card: 国政調査権（62条）の限界として、司法権との関係で許されない調査とはどのようなものか？

## Question
国政調査権（62条）の限界として、司法権との関係で許されない調査とはどのようなものか？

## Answer
- 係属中の裁判に対する事実認定や量刑の調査。
- 裁判官の心証に影響を与えるような批判的調査。
- 訴訟指揮への介入。

 **趣旨** ：国会の権能と「司法権の独立」の調整を図るため。 
 **条文** ：憲法62条（国政調査権）、76条3項（司法権の独立） 
 **あてはめ** ： **浦和充子事件** において、国会が量刑の当否を批判・調査したことは、司法権の独立を侵害するものとして最高裁から強く抗議された。
