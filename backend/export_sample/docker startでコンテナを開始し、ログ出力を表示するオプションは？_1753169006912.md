# Card: docker startでコンテナを開始し、ログ出力を表示するオプションは？

## Question
docker startでコンテナを開始し、ログ出力を表示するオプションは？

## Answer
docker start -a<br>-a : "attach"（接続する）<br><pre><code>docker start -a my-app          # コンテナ開始してログを表示
docker start --attach webapp    # webappを開始してログを表示</code></pre>
