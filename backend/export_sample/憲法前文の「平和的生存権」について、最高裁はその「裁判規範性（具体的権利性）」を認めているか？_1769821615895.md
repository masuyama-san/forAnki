# Card: 憲法前文の「平和的生存権」について、最高裁はその「裁判規範性（具体的権利性）」を認めているか？

## Question
憲法前文の「平和的生存権」について、最高裁はその「裁判規範性（具体的権利性）」を認めているか？

## Answer
認めていない。
※裁判の解釈基準にはなるが、直接訴えを起こせる具体的権利ではない

 **趣旨** : 憲法前文の理念的性格と、裁判上の権利としての限界を画定するため。
 **解説** : 最高裁は、平和的生存権について「すべての司法判断において、その解釈の基準となる」としつつも、それ自体を根拠として国に対して差止請求や損害賠償請求ができるような「具体的権利」性は否定している（抽象的権利説に近い立場）。したがって、平和的生存権侵害のみを理由とする訴えは認められない。
 **条文** : 憲法前文
