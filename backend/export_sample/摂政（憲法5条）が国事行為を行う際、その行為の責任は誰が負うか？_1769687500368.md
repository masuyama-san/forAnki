# Card: 摂政（憲法5条）が国事行為を行う際、その行為の責任は誰が負うか？

## Question
摂政（憲法5条）が国事行為を行う際、その行為の責任は誰が負うか？

## Answer
内閣

 **趣旨** : 摂政の行為も天皇の行為と同様に、政治的責任を内閣に帰属させることで、天皇制の非政治性を維持するため。 
 **解説** : 憲法5条は、摂政が「天皇の名でその国事に関する行為を行ふ」と定め、4条1項（国政に関する権能の否定）を準用している。摂政による行為も、憲法3条の「天皇の国事に関するすべての行為」に含まれると解され、内閣の助言と承認が必要となる。したがって、その行為の結果生じる政治的・法的責任は、すべて内閣が引き受けることになる。 
 **条文** : 憲法5条、3条
