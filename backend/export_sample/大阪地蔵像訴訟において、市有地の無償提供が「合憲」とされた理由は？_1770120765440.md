# Card: 大阪地蔵像訴訟において、市有地の無償提供が「合憲」とされた理由は？

## Question
大阪地蔵像訴訟において、市有地の無償提供が「合憲」とされた理由は？

## Answer
地蔵像の管理や祭祀が、宗教的活動というより**「地域の伝統的習俗（世俗的行事）」**としての色彩が強かったから。

 **趣旨** : 施設の「宗教性」の程度による判断の分かれ目。
 **解説** : 空知太神社（神道）や孔子廟（儒教）は宗教性が強いとされたが、路上の地蔵像は地域住民の町内会活動・伝統行事としての側面が強く、特定の宗教への援助・助長にはあたらないとされた。この「習俗か宗教か」の認定が結論を分ける。
 **条文** : 憲法20条、89条
 **出典** : 大阪地蔵像訴訟（最判平4.11.16）
