# Card: 戸別訪問の一律禁止（公職選挙法）について、最高裁は「合憲限定解釈」の手法を用いたか？

## Question
戸別訪問の一律禁止（公職選挙法）について、最高裁は「合憲限定解釈」の手法を用いたか？

## Answer
**用いていない** 。

（一律禁止の規定そのものを「必要かつ合理的」として合憲とした）

 **趣旨** : 選挙の自由・公正、買収防止、生活の平穏の確保。
 **解説** :  税関検査事件では「合憲限定解釈」が用いられたが、戸別訪問禁止事件では「一律禁止は広すぎる」という批判に対し、限定解釈を採用せず、禁止により得られる利益が失われる利益より大きいとして、規定全体を合憲とした。
 **条文** : 憲法21条1項
 **出典** : 最判昭56.7.21（※最大判昭56.6.15の法理を踏襲）
