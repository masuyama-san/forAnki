# Card: 企業が労働者の採用にあたり、特定の思想・信条を有することを理由に「採用を拒否」することは違法（公序良

## Question
企業が労働者の採用にあたり、特定の思想・信条を有することを理由に「採用を拒否」することは違法（公序良俗違反）か？

## Answer
**原則として適法** （違法ではない）。 
企業には「 **採用の自由** 」があるため。

 **趣旨** : 経済活動の一環として、企業（私人）側にも契約締結の自由が保障されるため。
 **解説** : 三菱樹脂事件判決。企業は自己の営業活動のために誰を雇うかを決定する自由を有し、特定の思想信条を有する者を雇い入れないとしても、直ちに違法とはならない。憲法14条（平等権）などは直接適用されない。
 **条文** : 憲法22条・29条、民法90条
 **出典** : 最大判昭48.12.12（三菱樹脂事件）
