# Card: 尊属殺重罰規定（刑法旧200条）違憲判決において、違憲とされた決定的な理由は？

## Question
尊属殺重罰規定（刑法旧200条）違憲判決において、違憲とされた決定的な理由は？

## Answer
立法目的（尊属の尊重）は合憲だが、 **手段（死刑・無期懲役のみ）が著しく均衡を失し不合理** である。

 **趣旨** : 刑罰における平等を確保し、過酷すぎる制約を排除するため。
 **解説** : 日本初の法令違憲判決。判例は、尊属に対する道徳的義務を刑法で保護すること自体（目的）は認めたが、その刑罰があまりに重く、酌量減軽しても執行猶予が付かない点は、手段として必要最小限を超え **著しく不合理** であるとした。
 **条文** : 憲法14条1項、旧刑法200条
 **出典** : 最大判昭48.4.4
