# Card: 世界で初めて、自然権思想に基づく人権を「成文憲法（権利章典）」として実定化した文書は何か？

## Question
世界で初めて、自然権思想に基づく人権を「成文憲法（権利章典）」として実定化した文書は何か？

## Answer
バージニア権利章典（1776年・米）

 **趣旨** : 抽象的な自然法思想を、法的拘束力のある文書へと転換させた歴史的画期性を押さえるため。
 **解説** : アメリカ独立宣言（1776年7月）やフランス人権宣言（1789年）よりも早く、同年6月に制定された。「人は生まれながらにして自由かつ独立」と宣言し、革命権や権力分立も明記されている。近代立憲主義の幕開けを告げる文書として、短答式試験の頻出知識である。
 **条文** : バージニア権利章典 第1条
