# Card: 公務員の社会的評価を低下させる論評が、名誉毀損の不法行為とならないための要件（公正な論評の法理）は？

## Question
公務員の社会的評価を低下させる論評が、名誉毀損の不法行為とならないための要件（公正な論評の法理）は？

## Answer
- その論評が専ら **公益目的** でなされたこと。
- 前提となる事実が、主要な点において **真実であるとの証明** があること。

 **趣旨** : 公務員の行動は公的関心事であり、自由な批判・監視の下に置かれるべきだから。
 **解説** : 長崎教師ビラ事件（正当防衛の成否ではなく、不法行為の成否を論じた事例）。「意見・論評」による名誉毀損の場合、前提事実さえ真実であれば、その評価がどんなに辛辣であっても、人身攻撃に及ぶなど論評の域を逸脱しない限り違法性が阻却される。
 **条文** : 民法709条、刑法230条の2（法理の参照）
 **出典** : 長崎教師ビラ事件（最判平元.12.21）
