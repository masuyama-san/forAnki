# Card: 憲法19条「思想・良心の自由」において、判例は「思想・良心」の定義をどのように解しているか？

## Question
憲法19条「思想・良心の自由」において、判例は「思想・良心」の定義をどのように解しているか？

## Answer
事物の是非弁別に関する判断（思想）および、その判断に基づく内面的規範（良心）。
区別の有無: 「思想」と「良心」を厳密に区別せず、一体として保障する（非区別説）。
対象: 宗教的信条に限らず、世俗的な世界観や主義主張も広く含む。

 **趣旨** : 個人の人格形成の核心である内面領域を、国家権力の介入から絶対的に保護するため。
 **解説** : 学説には信仰に準じる世界観のみを保護する「限定説」もあるが、判例は広く世俗的な思想も含むと解している。ただし、単なる事実の知不知（例：住所を知っているか）は含まれない。
 **条文** : 憲法19条
 **出典** : 謝罪広告強制事件（最大判昭31.7.4）
