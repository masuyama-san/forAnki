# Card: 一元的内在制約説に対する批判として、「条文の意味が希薄になる」以外に挙げられる、実務上の最大の弱点は

## Question
一元的内在制約説に対する批判として、「条文の意味が希薄になる」以外に挙げられる、実務上の最大の弱点は何か？

## Answer
基準が抽象的であり、具体的な合憲性判定の指標（ものさし）にならない点。

 **趣旨** : 裁判官の主観的評価による不安定な判決を避けるため。
 **解説** : 「公平の原理」という定義は哲学的で正しいが、実際の裁判で「この規制は公平か？」と問うても答えが出にくい。この抽象性を克服するために、後に「違憲審査基準論（二重の基準論）」が導入され、内在制約説とセットで運用されるようになった。
 **条文** : 憲法81条
