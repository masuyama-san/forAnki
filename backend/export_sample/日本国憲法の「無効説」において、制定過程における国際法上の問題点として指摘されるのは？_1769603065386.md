# Card: 日本国憲法の「無効説」において、制定過程における国際法上の問題点として指摘されるのは？

## Question
日本国憲法の「無効説」において、制定過程における国際法上の問題点として指摘されるのは？

## Answer
- 占領軍による内政不干渉原則の違反（押し付け憲法論）
- ハーグ陸戦条約43条違反

 **趣旨** : 占領下の制限された状況下で、占領軍が憲法の全面改訂を強要したことは、国際法上の正当性を欠くという主張。 
 **条文** : ハーグ陸戦条約43条（占領地の現行法の尊重義務）。 
 **判例・あてはめ** : 学説上は存在するが、実務・通説（有効説）は、サンフランシスコ平和条約の締結やその後の国民による追認をもって有効性を肯定する。
