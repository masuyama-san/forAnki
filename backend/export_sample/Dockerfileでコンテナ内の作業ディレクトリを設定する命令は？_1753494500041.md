# Card: Dockerfileでコンテナ内の作業ディレクトリを設定する命令は？

## Question
Dockerfileでコンテナ内の作業ディレクトリを設定する命令は？

## Answer
WORKDIR<br><pre><code>WORKDIR /app                       # /appディレクトリを作業ディレクトリに設定
WORKDIR /var/www/html              # Webアプリケーションディレクトリに設定
WORKDIR /home/user/project         # ユーザープロジェクトディレクトリに設定</code></pre>
