# Card: 憲法21条における「報道の自由」と「取材の自由」の保障の程度の違いは？

## Question
憲法21条における「報道の自由」と「取材の自由」の保障の程度の違いは？

## Answer
-  **報道の自由** ：憲法21条の **保障の下にある** 。
-  **取材の自由** ：憲法21条の **精神に照らし十分尊重** に値する（保障そのものではない）。

 **趣旨** : 報道は国民の「知る権利」に奉仕する表現行為そのものだが、取材はその準備行為にすぎず、一般的な情報収集との区別が曖昧であるため。
 **解説** : 最高裁は、「報道」を事実を編集し発表する行為として21条で保障されるとした一方、「取材」については、表現の自由の不可欠の前提であることを認めつつも、「精神に照らし十分尊重」という表現にとどめた。この区別は、取材に対する制約（証拠提出命令等）の合憲性判定において、報道そのものへの制約よりも緩やかな基準を適用する根拠となる。
 **条文** : 憲法21条1項
 **出典** : 博多駅テレビフィルム提出命令事件（最大決昭44.11.26）
