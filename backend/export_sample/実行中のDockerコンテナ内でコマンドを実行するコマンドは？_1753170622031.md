# Card: 実行中のDockerコンテナ内でコマンドを実行するコマンドは？

## Question
実行中のDockerコンテナ内でコマンドを実行するコマンドは？

## Answer
docker exec<br><pre><code>docker exec my-container ls -la     # my-containerでls -laを実行
docker exec webapp ps aux           # webappコンテナでプロセス一覧を表示
docker exec database mysql --version # databaseコンテナでMySQLバージョン確認</code></pre>
