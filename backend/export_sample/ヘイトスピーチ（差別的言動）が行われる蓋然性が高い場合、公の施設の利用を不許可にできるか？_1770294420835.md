# Card: ヘイトスピーチ（差別的言動）が行われる蓋然性が高い場合、公の施設の利用を不許可にできるか？

## Question
ヘイトスピーチ（差別的言動）が行われる蓋然性が高い場合、公の施設の利用を不許可にできるか？

## Answer
**許される場合がある** 。

差別的言動により、他者の人格権侵害等の「明らかな差し迫った危険」が具体的に予見される場合。

 **趣旨** : 表現の自由といえども、他者の尊厳や人格権を不当に侵害する自由まで保障するものではないから。
 **解説** : 従来、表現内容に基づく規制は慎重であるべきとされたが、ヘイトスピーチ解消法等の制定を受け、近時の裁判例（川崎市ヘイトデモ事件等）は、物理的暴力だけでなく「差別的言動による人格権侵害」も泉佐野基準の「危険」に該当すると判断している。
 **条文** : 憲法21条1項、ヘイトスピーチ解消法
 **出典** : 横浜地川崎支部決令5.7.11
