# Card: 憲法21条2項後段の「通信の秘密」は、通信の内容（本文）以外に何を保護するか？

## Question
憲法21条2項後段の「通信の秘密」は、通信の内容（本文）以外に何を保護するか？

## Answer
**通信の構成要素（メタデータ）** 。

具体的には、通信当事者の **氏名・住所** 、通信の **日時** 、 **個数** 、発着信場所など。

 **趣旨** : 「誰といつ通信したか」という事実自体が、個人の人間関係や思想信条を推知させるプライバシーの核心情報だから。
 **解説** : 封筒の中身や通話内容だけでなく、宛名、消印、通話ログ、IPアドレスなどの「構成要素」も、内容と同様に厳格に保護される。これらを公権力が取得する場合（捜査など）は、原則として令状が必要となる。
 **条文** : 憲法21条2項後段、電気通信事業法4条
 **出典** : 最判平11.12.16
