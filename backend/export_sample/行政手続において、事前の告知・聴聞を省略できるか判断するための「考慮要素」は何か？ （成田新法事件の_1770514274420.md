# Card: 行政手続において、事前の告知・聴聞を省略できるか判断するための「考慮要素」は何か？ （成田新法事件の

## Question
行政手続において、事前の告知・聴聞を省略できるか判断するための「考慮要素」は何か？
（成田新法事件の判断枠組み）

## Answer
-  **行政処分の性質・内容** 
-  **被侵害利益の程度** 
-  **行政目的達成の緊急性・必要性**  
これらを総合考慮（スライディング・スケール）して決定する。

 **趣旨** : 行政活動は多種多様であり、一律に厳格な事前手続を課すと、かえって公益（行政目的）を阻害するおそれがあるため。
 **解説** : 「権利侵害が重大」なら厳格な手続が必要ですが、「緊急性が高い（成田新法事件）」や「大量処理が必要（交通反則金）」な場合は、事後の救済手続で足りるとされるなど、柔軟に解釈されます。
 **条文** : 憲法31条（の法意）
 **出典** : 最大判平4.7.1（成田新法事件）
