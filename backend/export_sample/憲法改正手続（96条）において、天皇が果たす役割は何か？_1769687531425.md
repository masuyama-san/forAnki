# Card: 憲法改正手続（96条）において、天皇が果たす役割は何か？

## Question
憲法改正手続（96条）において、天皇が果たす役割は何か？

## Answer
改正の公布（国民の名で、内閣の助言と承認により行う）

 **趣旨** : 国民投票によって成立した憲法改正を、国家の形式的な代表者である天皇が公に公表するため。 
 **解説** : 天皇は憲法改正が成立した際、その「公布」を行う（7条1号）。この公布は「国民の名で」行われることが96条2項に明記されており、主権者たる国民の意思が最高法規を確定させたことを形式的に整える役割を担う。ここでも実質的な内容決定権はなく、内閣の助言と承認が要件となる。公布という形式的行為により、改正の効力発生プロセスが完了する。 
 **条文** : 憲法96条2項、7条1号
