# Card: 天皇が国事行為を臨時に代行させる「委任」を行う際、必要となる手続は何か？

## Question
天皇が国事行為を臨時に代行させる「委任」を行う際、必要となる手続は何か？

## Answer
- 内閣の助言と承認
- 法律に基づく委任（国事行為の臨時代行に関する法律）

 **趣旨** : 天皇が病気等で国事行為を行えない場合でも、国家の形式的機能を維持しつつ、その行為を内閣のコントロール下に置くため。 
 **解説** : 天皇は、精神・身体の疾患や事故がある場合、摂政を置くべき事態を除き、内閣の助言と承認により国事行為を委任できる（憲法4条2項）。この詳細は「国事行為の臨時代行に関する法律」に定められており、委任を受けた皇族は、その間は刑事訴追を受けない（同法6条）。摂政（5条）が「制度」として天皇の名で行為するのに対し、臨時代行は天皇による「委任」に基づく点が異なる。 
 **条文** : 憲法4条2項
