# Card: docker logsの出力をファイルに保存する方法は？

## Question
docker logsの出力をファイルに保存する方法は？

## Answer
docker logs コンテナ名 &gt; ファイル名<br><pre><code>docker logs webapp &gt; webapp.log              # ログをファイルに保存
docker logs --since "1d" app &gt;&gt; daily.log    # 1日分のログを追記保存
docker logs -t webapp 2&gt;&amp;1 | tee output.log  # 画面表示しながらファイル保存</code></pre>
