# Card: Dockerコンテナのログを表示するコマンドは？

## Question
Dockerコンテナのログを表示するコマンドは？

## Answer
docker logs<br><pre><code>docker logs my-container        # my-containerのログを表示
docker logs c1d2e3f4             # コンテナIDでログ表示
docker logs webapp              # webappコンテナのログを表示</code></pre>
