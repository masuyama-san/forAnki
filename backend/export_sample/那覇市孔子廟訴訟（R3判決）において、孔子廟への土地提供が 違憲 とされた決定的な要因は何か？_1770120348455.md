# Card: 那覇市孔子廟訴訟（R3判決）において、孔子廟への土地提供が 違憲 とされた決定的な要因は何か？

## Question
那覇市孔子廟訴訟（R3判決）において、孔子廟への土地提供が **違憲** とされた決定的な要因は何か？

## Answer
使用料の**全額免除（無償提供）**を行ったこと。

 **趣旨** : 公有財産を用いた特定の宗教への経済的援助の禁止。
 **解説** : 孔子廟が観光資源としての側面を持つことは認めたが、宗教的性格は否定できないとした。その上で、使用料を全額免除することは、一般人の目から見て「市が特定の宗教を特別に支援している」と映るため、政教分離原則が許容する限度を超えると判断された。
 **条文** : 憲法20条、89条
 **出典** : 那覇市孔子廟敷地使用料免除無効確認請求事件（最大判令3.2.24）
