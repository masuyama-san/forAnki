# Card: 京都府学連事件において、警察官による個人の容貌・姿態の撮影（肖像権侵害の有無）が適法とされるための3

## Question
京都府学連事件において、警察官による個人の容貌・姿態の撮影（肖像権侵害の有無）が適法とされるための3要件は？

## Answer
-  **現行犯性** （現に犯罪が行われている、または行われた直後であること）。
-  **必要性・緊急性** （証拠保全の必要があり、かつ緊急であること）。
-  **相当性** <span style="font-weight: bold;">（撮影方法が一般的に許容される限度を超えないこと）。</span>

 **** 
 **趣旨** : 個人の肖像権（みだりに撮影されない自由）と、公共の利益（犯罪捜査）の調整。
 **解説** : デモ行進中の撮影について、公道上の行為であっても肖像権の保護が及ぶとしつつ、上記要件を満たす場合は「正当な理由」があり合憲とした。「相当性」とは、例えば執拗な追尾撮影や隠し撮りなど、不当な方法でないことを指す。
 **条文** : 憲法13条、35条
 **出典** : 最大判昭44.12.24（京都府学連事件）
