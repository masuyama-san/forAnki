# Card: 憲法19条に対する「直接的制約」と「間接的制約」の違いは？

## Question
憲法19条に対する「直接的制約」と「間接的制約」の違いは？

## Answer
-  **直接的制約** : 特定の思想の強制・禁止そのもの（踏み絵など）。原則として **絶対的に禁止（違憲）** 。
-  **間接的制約** : 行為自体は中立だが、結果として個人の思想と葛藤を生じさせるもの。 **合理的関連性の基準** （利益衡量）で判断する。

 **趣旨** : 制約の態様に応じた審査基準を使い分け、公共の福祉との調整を図るため。
 **解説** : 「君が代ピアノ伴奏拒否事件」の重要ロジック。伴奏命令自体は「職務命令」であり思想の表明ではないが、個人の歴史観等と矛盾する場合（間接的制約）、職務の公共性や必要性と、個人の不利益を比較衡量し、許容限度内であれば合憲とされる。
 **条文** : 憲法19条、12条
 **出典** : 君が代ピアノ伴奏職務命令拒否事件（最判平19.2.27）
