# Card: 私立大学においても、憲法23条の「学問の自由・大学の自治」は保障されるか？

## Question
私立大学においても、憲法23条の「学問の自由・大学の自治」は保障されるか？

## Answer
**保障される。** 

真理の探究という大学の本質は、国公立か私立かを問わないため。

 **趣旨** : 学問研究機関としての共通性。
 **解説** : 判例もポポロ事件において、大学の公私を区別せず自治を論じている。ただし、私立大学は独自の「建学の精神」を持つため、その範囲内で宗教教育を行う自由なども認められる。
 **条文** : 憲法23条
