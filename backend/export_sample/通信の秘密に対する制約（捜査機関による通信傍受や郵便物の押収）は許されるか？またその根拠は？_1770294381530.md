# Card: 通信の秘密に対する制約（捜査機関による通信傍受や郵便物の押収）は許されるか？またその根拠は？

## Question
通信の秘密に対する制約（捜査機関による通信傍受や郵便物の押収）は許されるか？またその根拠は？

## Answer
許される場合がある。

根拠は **公共の福祉** であり、法律の規定と**厳格な司法審査（令状主義）**が必要となる。

 **趣旨** : 通信の秘密も絶対無制約ではなく、重大犯罪の捜査など、社会共同生活上の極めて重要な利益との調整が必要だから。
 **解説** : 「検閲」は絶対禁止だが、「通信の秘密」は相対的な禁止である。しかし、表現の自由やプライバシー権と直結するため、制約が許されるのは必要最小限の場合（重大犯罪等）に限られ、通信傍受法のように極めて厳格な手続的要件が課される。
 **条文** : 憲法21条2項、35条
 **出典** : 最決平11.12.16
