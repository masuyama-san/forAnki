# Card: 参議院の定数配分において、衆議院よりも広い裁量（大きな格差）が許容される根拠とされる概念は？

## Question
参議院の定数配分において、衆議院よりも広い裁量（大きな格差）が許容される根拠とされる概念は？

## Answer
**地域代表的性格**  （都道府県を選挙区の単位とすることの意義）

 **趣旨** : 二院制の下、参議院には衆議院（人口比例原則が徹底される）とは異なる独自性が認められるため。
 **解説** : 判例は、参議院について「都道府県を単位とする地域代表的な性格」を加味することを認めている。ただし、近時の判例は投票価値の平等を重視する傾向を強めており、かつてほど大きな格差は許容されなくなっている点に留意。
 **条文** : 憲法14条、43条
 **出典** : 最大判平24.10.17 等
