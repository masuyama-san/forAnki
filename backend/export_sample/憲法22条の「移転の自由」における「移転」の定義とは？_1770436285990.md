# Card: 憲法22条の「移転の自由」における「移転」の定義とは？

## Question
憲法22条の「移転の自由」における「移転」の定義とは？

## Answer
住所または居所を変更する自由（一時的な国内旅行も含む）。

 **趣旨** : 生活の本拠（住所）の変更だけでなく、移動そのものも保障範囲内である。
 **解説** : 「移転」とは、住所・居所の変更を意味するが、これには永続的な変更のみならず、一時的な移動（旅行）も含まれると解される。移動の自由は、経済活動や表現の自由の実効性を確保するための手段的権利としての性格を持つ。国内旅行は1項、海外旅行は2項（解釈）で保障される。
 **条文** : 憲法22条1項
