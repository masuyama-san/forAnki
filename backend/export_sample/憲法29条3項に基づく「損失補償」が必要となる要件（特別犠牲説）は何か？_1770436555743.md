# Card: 憲法29条3項に基づく「損失補償」が必要となる要件（特別犠牲説）は何か？

## Question
憲法29条3項に基づく「損失補償」が必要となる要件（特別犠牲説）は何か？

## Answer
①特定の人に対する制約であり、かつ、②財産権の本質的侵害（受忍限度を超える侵害）であること

 **趣旨** : 公共のために特定の個人だけが犠牲になることは、平等の原則（14条）に反するため、金銭による負担の公平化を図る必要がある。
 **解説** : 損失補償の要否は「特別犠牲説」によって判断されるのが通説・判例である。①形式的要件として、一般人への普遍的な制約ではなく特定人への制約であること、②実質的要件として、その制約が財産権の利用・処分を著しく妨げ、社会通念上の受忍限度を超えていること、の双方が必要とされる。
 **条文** : 憲法29条3項
