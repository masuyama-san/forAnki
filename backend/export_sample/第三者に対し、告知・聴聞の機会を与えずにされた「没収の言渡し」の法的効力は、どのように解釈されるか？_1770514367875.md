# Card: 第三者に対し、告知・聴聞の機会を与えずにされた「没収の言渡し」の法的効力は、どのように解釈されるか？

## Question
第三者に対し、告知・聴聞の機会を与えずにされた「没収の言渡し」の法的効力は、どのように解釈されるか？

## Answer
没収の言渡し自体は有効だが、 **第三者の所有権を剥奪する効果を生じない** 。

（被告人に対する占有剥奪の効果のみ生じる）

 **趣旨** : 違憲な状態を回避しつつ、既に確定した判決の形式的有効性を維持するための「合憲限定解釈」。
 **解説** : 判決自体を無効・取消しにするのではなく、「所有権移転効が生じない」と解します。つまり、所有権は第三者に留保されたままであり（国に移転しない）、被告人に対しては「占有を奪われる（使用できない）」という効果のみが残存します。
 **条文** : 憲法31条
 **出典** : 最大判昭37.11.28（第三者所有物没収事件）
