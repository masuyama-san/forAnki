# Card: 天皇には裁判所の民事裁判権が及ぶか？

## Question
天皇には裁判所の民事裁判権が及ぶか？

## Answer
及ばない（民事裁判権は否定される）。

 **趣旨** : 天皇の「象徴」としての地位の特殊性を保護し、訴訟を通じた不当な政治的影響を避けるため。 
 **解説** : 最判平元・11・20（記帳所事件）は、天皇が日本国および日本国民統合の象徴であることに鑑み、その地位の特殊性から天皇には民事裁判権が及ばないとした。もし民事裁判権を認めれば、天皇が被告として出頭したり、証人義務を負ったりすることになり、象徴としての地位と相容れないと考えられた。このため、天皇を被告とする訴えは、不適法として訴状が却下される。 
 **条文** : 憲法1条、裁判所法3条
