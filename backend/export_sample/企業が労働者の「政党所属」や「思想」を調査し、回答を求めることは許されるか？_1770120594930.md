# Card: 企業が労働者の「政党所属」や「思想」を調査し、回答を求めることは許されるか？

## Question
企業が労働者の「政党所属」や「思想」を調査し、回答を求めることは許されるか？

## Answer
**許される（適法）。** 

企業活動の維持のため、必要かつ合理的な範囲内の調査であれば違法性はない。

 **趣旨** : 企業の組織秩序維持とプライバシーの調整。
 **解説** :  **採用後** の調査に関する判例。企業は労働力として適格かどうかを判断するために調査を行う権限がある。ただし、その調査結果に基づいて **差別的な取り扱い** （解雇や昇進差別など）をすることは、公序良俗（民法90条）や労基法3条違反として違法となる。「調査はOK、差別はNG」という区別が重要。
 **条文** : 憲法19条、民法90条
 **出典** : 東電塩山営業所事件（最判昭63.2.5）
