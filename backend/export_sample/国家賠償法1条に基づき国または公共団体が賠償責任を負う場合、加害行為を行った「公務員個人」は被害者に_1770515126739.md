# Card: 国家賠償法1条に基づき国または公共団体が賠償責任を負う場合、加害行為を行った「公務員個人」は被害者に

## Question
国家賠償法1条に基づき国または公共団体が賠償責任を負う場合、加害行為を行った「公務員個人」は被害者に対して賠償責任を負うか？

## Answer
負わない。

（責任は国等に限定される＝絶対的免責）

 **趣旨** : 公務員個人が過大な責任を恐れて職務遂行を躊躇することを防ぐとともに、資力のある国が賠償することで被害者救済を図るため。
 **解説** : 判例は、国家賠償法が適用される場合、公務員個人は被害者に対して直接の民事上の損害賠償責任（不法行為責任）を負わないと解している。なお、公務員に「故意または重過失」がある場合に限り、国は賠償後にその公務員に対して求償権を行使することができる（国賠法1条2項）。
 **条文** : 国家賠償法1条
 **出典** : 最判昭30.4.19
