# Card: ハンセン病患者を強制的に療養所へ入所・隔離させていた旧らい予防法の規定について、裁判所（熊本判決）は

## Question
ハンセン病患者を強制的に療養所へ入所・隔離させていた旧らい予防法の規定について、裁判所（熊本判決）はどのように判断したか？

## Answer
違憲（国家賠償請求を認容）。

 **趣旨** : 医学的知見の変遷により隔離の必要性が失われたにもかかわらず、漫然と拘束を続けたことは正当化できない。
 **解説** : 居住・移転の自由の制約が、実質的に**「人身の自由（身体の拘束）」**や人格権の侵害に及ぶ場合、極めて厳格に審査される例。単なる場所の移動の制限を超え、人生の機会を奪う重大な人権侵害と認定された。経済的自由の制約とは次元の異なる問題として処理される。
 **条文** : 憲法22条、13条、国家賠償法1条
 **出典** : 熊本地判平13.5.11
