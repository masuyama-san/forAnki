# Card: 1215年のマグナ・カルタ（大憲章）で確認された権利は、「近代的意味の人権」といえるか？

## Question
1215年のマグナ・カルタ（大憲章）で確認された権利は、「近代的意味の人権」といえるか？

## Answer
いえない。 
貴族などが王権に対して有する封建的な「特権（Privilege）」であり、普遍的な「人権」ではないから。

 **趣旨** : 「人権」概念の歴史的非連続性を理解し、前近代的な身分特権と、近代の天賦人権を明確に区別するため。
 **解説** : マグナ・カルタは「法の支配」の源流として重要だが、その実態は諸侯が王から勝ち取った既得権益の確認である。「人間であれば誰でも生まれながらに有する」という自然権思想（天賦人権説）に基づかないため、これをもって「近代立憲主義の確立」とするのは誤りである。
 **条文** : マグナ・カルタ（1215年）
