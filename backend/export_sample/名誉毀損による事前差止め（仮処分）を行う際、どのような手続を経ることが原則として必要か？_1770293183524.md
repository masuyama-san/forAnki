# Card: 名誉毀損による事前差止め（仮処分）を行う際、どのような手続を経ることが原則として必要か？

## Question
名誉毀損による事前差止め（仮処分）を行う際、どのような手続を経ることが原則として必要か？

## Answer
**口頭弁論** または **債務者の審尋** 。

（相手方の言い分を聞く手続）

 **趣旨** : 表現の自由を制約する以上、手続的保障（Due Process）を徹底し、不当な事前抑制を防ぐため。
 **解説** : 北方ジャーナル事件判決は実体要件だけでなく手続要件も示した。密行性が求められる通常の仮処分とは異なり、表現の自由案件では、原則として表現者側（債務者）に反論の機会を与えなければ、憲法21条の趣旨に反し許されないとした。
 **条文** : 民事保全法、憲法21条
 **出典** : 北方ジャーナル事件（最大判昭61.6.11）
