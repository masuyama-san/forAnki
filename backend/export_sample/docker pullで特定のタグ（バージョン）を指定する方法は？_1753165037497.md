# Card: docker pullで特定のタグ（バージョン）を指定する方法は？

## Question
docker pullで特定のタグ（バージョン）を指定する方法は？

## Answer
docker pull イメージ名:タグ<br><pre><code>docker pull python:3.9         # Python 3.9をダウンロード
docker pull node:16-alpine     # Node.js 16のAlpine版をダウンロード
docker pull redis:6.2          # Redis 6.2をダウンロード</code></pre>
