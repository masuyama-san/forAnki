# Card: 私人間効力の問題において、人権侵害行為が「法律行為（契約・解雇・就業規則等）」である場合、その効力を

## Question
私人間効力の問題において、人権侵害行為が「法律行為（契約・解雇・就業規則等）」である場合、その効力を否定するために用いる民法の条文は？

## Answer
民法90条（公序良俗）

 **趣旨** : 社会的妥当性を欠く法律行為の私法上の効力（有効・無効）を判断するため。
 **解説** : 契約や解雇などの「法律行為」の内容が、憲法の人権保障の趣旨に反し、社会的に許容しうる限界を超える場合、公序良俗違反として「無効」とする。
 **条文** : 民法90条
 **出典** : 最判昭56.3.24（日産自動車事件）
