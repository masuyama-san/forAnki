# Card: 実行中のDockerコンテナ一覧を表示するコマンドは？

## Question
実行中のDockerコンテナ一覧を表示するコマンドは？

## Answer
docker ps<br><pre><code>docker ps                       # 実行中のコンテナ一覧を表示
docker ps -q                    # コンテナIDのみ表示
docker ps --format "table {{.Names}}\t{{.Status}}"  # カスタム形式</code></pre>
