# Card: 労働組合 が、特定の候補者を支持する等の「政治活動（統一候補決定）」を行うことは許されるか？

## Question
**労働組合** が、特定の候補者を支持する等の**「政治活動（統一候補決定）」**を行うことは許されるか？

## Answer
許される（自由）。

 **趣旨** : 労働組合の活動の自由（団結権の派生）として、目的達成のために広範な活動が認められる。
 **解説** : 労働組合は、経済的地位の向上という目的のために、組織として特定の政党や候補者を支持決定し、選挙運動を推進することは自由である（目的の範囲内）。組合員の政治的自由との関係でも、勧告や説得の範囲であれば適法とされる。
 **条文** : 憲法21条、28条
