# Card: 久米至聖廟設置許可訴訟（R7判決）において、同一施設への関わりが 合憲 とされた理由は何か？

## Question
久米至聖廟設置許可訴訟（R7判決）において、同一施設への関わりが **合憲** とされた理由は何か？

## Answer
使用料を**徴収（有償）**した上で設置許可を与えたこと。

 **趣旨** : 対価（使用料）の有無により、特定の宗教への「援助性」が否定されるため。
 **解説** : 画像資料の重要ポイント。R3判決は「免除（無償）」を違憲としたが、R7判決は「許可・徴収（有償）」を合憲とした。一般社団法人に対し、適正な対価を求めて場所を貸すだけならば、他の一般利用者と同等の扱いであり、特別な便益供与（援助・助長）にはあたらない。
 **条文** : 憲法20条、89条
 **出典** : 最判令7.3.17
