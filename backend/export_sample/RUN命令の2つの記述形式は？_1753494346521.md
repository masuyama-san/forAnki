# Card: RUN命令の2つの記述形式は？

## Question
RUN命令の2つの記述形式は？

## Answer
Shell形式：RUN コマンド <br>Exec形式：RUN ["実行ファイル", "引数1", "引数2"]<br><pre><code># Shell形式（一般的）
RUN apt-get update &amp;&amp; apt-get install -y curl

# Exec形式
RUN ["apt-get", "update"]
RUN ["/bin/bash", "-c", "echo hello"]</code></pre>
