# Card: 前科照会事件において、市区町村長が漫然と弁護士会照会に応じて前科を報告した行為の適法性は？

## Question
前科照会事件において、市区町村長が漫然と弁護士会照会に応じて前科を報告した行為の適法性は？

## Answer
**違法** （公権力の違法な行使にあたり、国家賠償責任を負う）。

 **趣旨** : 前科情報は社会復帰（更生）に関わる極めてセンシティブな情報であるため。
 **解説** : 前科等のある者も「みだりに公開されない法律上の保護に値する利益」を有する。市区町村長は照会の必要性と個人の不利益を慎重に比較衡量する義務があり、漫然と回答したことは過失による違法行為となる。
 **条文** : 憲法13条、国家賠償法1条
 **出典** : 最判昭56.4.14
