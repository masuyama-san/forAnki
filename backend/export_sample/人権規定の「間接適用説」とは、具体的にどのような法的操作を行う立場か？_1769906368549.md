# Card: 人権規定の「間接適用説」とは、具体的にどのような法的操作を行う立場か？

## Question
人権規定の「間接適用説」とは、具体的にどのような法的操作を行う立場か？

## Answer
**私法の一般条項（民法90条、709条等）の解釈・適用** において、憲法の趣旨を判断基準として読み込む立場。

 **趣旨** : 私的自治の維持と、社会的権力（企業等）による人権侵害の救済の調和を図るため。
 **解説** : 憲法違反を直接の無効事由とはせず、「憲法の趣旨に反するから、公序良俗違反（民法90条）となる」あるいは「違法性がある（民法709条）」というロジックで、あくまで私法上の効果として解決を図る。
 **条文** : 民法90条、709条
 **出典** : 最大判昭48.12.12（三菱樹脂事件）
