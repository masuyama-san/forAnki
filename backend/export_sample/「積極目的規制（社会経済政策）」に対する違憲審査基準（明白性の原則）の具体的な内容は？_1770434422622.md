# Card: 「積極目的規制（社会経済政策）」に対する違憲審査基準（明白性の原則）の具体的な内容は？

## Question
「積極目的規制（社会経済政策）」に対する違憲審査基準（明白性の原則）の具体的な内容は？

## Answer
立法府の判断が、その合理的裁量の範囲を逸脱し、著しく不合理であることが明白でない限り、合憲とする。

 **趣旨** : 誰を保護し経済をどう調整するかは高度な政治判断であり、裁判所は原則として介入すべきではない。
 **解説** : 積極目的（例：中小小売商保護）においては、どのような手段をとるかは国会の広い裁量に委ねられる。裁判所は、その判断が「一見して明白に不合理」といえない限り、合憲として判断を尊重する。
 **条文** : 憲法第22条第1項
 **出典** : 最大判昭47.11.22（小売市場距離制限事件）
