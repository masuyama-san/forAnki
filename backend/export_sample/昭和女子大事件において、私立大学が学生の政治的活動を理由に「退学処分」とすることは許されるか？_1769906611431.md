# Card: 昭和女子大事件において、私立大学が学生の政治的活動を理由に「退学処分」とすることは許されるか？

## Question
昭和女子大事件において、私立大学が学生の政治的活動を理由に「退学処分」とすることは許されるか？

## Answer
**原則として許される** （裁量権の範囲内）。

 **趣旨** : 私立大学独自の教育方針（建学の精神）と、学生の在学契約上の義務を尊重するため。
 **解説** : 私立大学は「建学の精神」に基づく独自の規律を設けることができ、学生はこれに服することを前提に入学している。したがって、合理的な学則違反に対する処分は、著しく妥当性を欠かない限り、大学の裁量権の範囲内として有効である。
 **条文** : 民法90条
 **出典** : 最判昭49.7.19（昭和女子大事件）
