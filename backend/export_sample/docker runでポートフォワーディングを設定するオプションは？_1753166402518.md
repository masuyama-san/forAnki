# Card: docker runでポートフォワーディングを設定するオプションは？

## Question
docker runでポートフォワーディングを設定するオプションは？

## Answer
docker run -p ホストポート:コンテナポート<br><code>-p：</code>"publish"（公開する）&nbsp;<br><pre><code>docker run -p 8080:80 nginx    # ホストの8080番をコンテナの80番に転送
docker run -p 3306:3306 mysql  # MySQLの標準ポート転送
docker run -p 80:80 -p 443:443 nginx  # 複数ポート転送</code></pre>
