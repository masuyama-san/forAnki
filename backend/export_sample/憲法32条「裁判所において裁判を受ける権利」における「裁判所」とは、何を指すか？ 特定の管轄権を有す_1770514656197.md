# Card: 憲法32条「裁判所において裁判を受ける権利」における「裁判所」とは、何を指すか？ 特定の管轄権を有す

## Question
憲法32条「裁判所において裁判を受ける権利」における「裁判所」とは、何を指すか？ 特定の管轄権を有する具体的な裁判所を指すか？

## Answer
憲法所定の「正規の裁判所」（76条）を指す。

（特定の管轄権を有する具体的裁判所までは指さない。）

 **趣旨** : 行政機関による終審裁判の禁止など、司法権の独立が確保された公正な機関による裁判を保障するためである。
 **解説** : 32条の「裁判所」とは、憲法76条に基づき構成され、裁判官の独立などが保障された組織としての裁判所を意味する。したがって、個別の事件について、ある特定の場所や種類の裁判所（例：東京地方裁判所）で審理を受ける権利までを不可変更のものとして保障するわけではない。それゆえ、合理的な理由で管轄を変更する法改正は合憲である。
 **条文** : 憲法32条、76条
 **出典** : 最大判昭24.3.23
