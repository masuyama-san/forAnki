# Card: 同日に判決が出た 堀越事件 （無罪）と 世田谷事件 （有罪）で、結論を分けた決定的な あてはめの要素

## Question
同日に判決が出た **堀越事件** （無罪）と **世田谷事件** （有罪）で、結論を分けた決定的な **あてはめの要素** は何か？

## Answer
管理職的地位（指揮命令権限）の有無

 **趣旨** : 指導的地位にある公務員の政治的偏向は、組織運営全体に影響を及ぼす蓋然性が高いため。
 **解説** : 世田谷事件の被告人は厚生労働省の「課長補佐（管理職）」であった。最高裁は、管理職が特定の政治的傾向を示すことは、指揮命令を通じて部下の職務や組織運営に影響を及ぼす「実質的なおそれ」があるとして、処罰を合憲とした。
 **条文** : 国家公務員法102条1項
