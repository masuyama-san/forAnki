# Card: 2014年（平成26年）の閣議決定および安保法制において、集団的自衛権の行使はどのように位置づけられ

## Question
2014年（平成26年）の閣議決定および安保法制において、集団的自衛権の行使はどのように位置づけられたか？

## Answer
「武力の行使の新3要件」を満たす場合に限り、自衛のための必要最小限度の実力行使として許容される（合憲）。

 **趣旨** : 安全保障環境の変化に対応するため、従来の「集団的自衛権の行使は違憲」という解釈を変更した。
 **解説** : 従来、政府は国際法上の集団的自衛権を有しているが行使はできないとしてきた。しかし、2014年の解釈変更により、密接な関係にある他国への攻撃により「わが国の存立が脅かされる明白な危険」がある場合等（新3要件）には、集団的自衛権の行使も憲法9条の下で許容される「必要最小限度の実力」の範囲内であると整理された。
 **条文** : 憲法9条
