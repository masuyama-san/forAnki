# Card: 「認証」とは、天皇が実質的な決定権を行使することを意味するか？

## Question
「認証」とは、天皇が実質的な決定権を行使することを意味するか？

## Answer
実質的決定権はない。
すでになされた行為の正当な手続を公に証明する形式的行為。

 **趣旨** : 天皇の権能を形式的・儀礼的なものに限定し、政治的影響力を排除するため。 
 **解説** : 天皇は憲法7条各号に掲げる行為を「認証」する。具体的には、国務大臣の任免（5号）や恩赦（6号）などが対象となる。認証とは、他の機関（内閣など）によって既になされた決定が正当な手続を経て行われたことを、天皇が公に証明する行為であり、天皇自身が内容を決定するものではない。したがって、天皇には内容を審査して拒絶する権限はなく、内閣の助言と承認に従う義務がある。 
 **条文** : 憲法7条
