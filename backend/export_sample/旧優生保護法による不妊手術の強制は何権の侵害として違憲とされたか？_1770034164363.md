# Card: 旧優生保護法による不妊手術の強制は何権の侵害として違憲とされたか？

## Question
旧優生保護法による不妊手術の強制は何権の侵害として違憲とされたか？

## Answer
**リプロダクティブ・ライツ**  （子を産むか産まないかを決定する自由）

 **趣旨** : 生殖に関する意思決定を、個人の尊厳と人格の尊重（13条）の中核と位置づけた。
 **解説** : 令和6年の最大判決。立法目的自体が「不良な子孫の出生防止」という優生思想に基づく差別的なものであり、正当性を欠くと断じた。国会議員の立法不作為責任（国家賠償法上の違法性）まで認められた画期的な判決である。
 **条文** : 憲法13条、国家賠償法1条1項
 **出典** : 最大判令6.7.3（旧優生保護法国家賠償請求事件）
