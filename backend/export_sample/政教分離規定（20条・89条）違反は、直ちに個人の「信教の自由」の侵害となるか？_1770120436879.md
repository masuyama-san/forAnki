# Card: 政教分離規定（20条・89条）違反は、直ちに個人の「信教の自由」の侵害となるか？

## Question
政教分離規定（20条・89条）違反は、直ちに個人の「信教の自由」の侵害となるか？

## Answer
**ならない。** 

政教分離規定は **制度的保障** であり、個人の権利を直接保障するものではないため（反射的利益）。

 **趣旨** : 信教の自由を確実に保障するための「防波堤（手段）」として、国家の非宗教性を客観的に定めた規定であるため。
 **解説** : したがって、国家が特定の宗教を援助した（政教分離違反）としても、それによって直ちに他の国民の信教の自由が侵害されたとはみなされない。政教分離違反のみを理由とする国家賠償請求や差止請求は、原則として認められない（損害の発生が必要）。
 **条文** : 憲法20条、89条
 **出典** : 箕面忠魂碑訴訟（最判平5.2.16）
