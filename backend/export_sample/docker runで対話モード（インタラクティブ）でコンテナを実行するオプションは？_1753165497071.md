# Card: docker runで対話モード（インタラクティブ）でコンテナを実行するオプションは？

## Question
docker runで対話モード（インタラクティブ）でコンテナを実行するオプションは？

## Answer
docker run -it<br>-i (--interactive) : 対話モードを有効にする<br>-t (--tty) : 擬似端末を割り当てる<br><pre><code>docker run -it ubuntu bash    # Ubuntuコンテナでbashを対話実行
docker run -it python         # Pythonの対話シェルを起動
docker run -it --rm alpine sh # Alpine Linuxで一時的にシェル実行</code></pre>
