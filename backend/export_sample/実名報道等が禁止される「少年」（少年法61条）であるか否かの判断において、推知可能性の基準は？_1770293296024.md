# Card: 実名報道等が禁止される「少年」（少年法61条）であるか否かの判断において、推知可能性の基準は？

## Question
実名報道等が禁止される「少年」（少年法61条）であるか否かの判断において、推知可能性の基準は？

## Answer
**不特定多数の一般人** が、その者を当該事件の本人であると推知することができるか。

 **趣旨** : 少年の更生と社会復帰を妨げないため、氏名・容貌等の報道を禁止する規定の実効性を確保するため。
 **解説** : 長良川リンチ事件。記事に実名がなくても、経歴、職業、住所などの記述を総合して、少年と面識のない「不特定多数の一般人」が本人だと特定できる場合は、少年法61条に違反し、プライバシー侵害として不法行為となる。
 **条文** : 少年法61条
 **出典** : 長良川リンチ殺人事件報道訴訟（最判平15.3.14）
