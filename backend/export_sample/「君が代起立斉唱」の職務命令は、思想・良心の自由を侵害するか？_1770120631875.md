# Card: 「君が代起立斉唱」の職務命令は、思想・良心の自由を侵害するか？

## Question
「君が代起立斉唱」の職務命令は、思想・良心の自由を侵害するか？

## Answer
**侵害しない（合憲）。** 

起立斉唱行為は、一般的・客観的に見て「国旗・国歌への敬意の表明」という **儀礼的な所作** であり、特定の思想の表明強制（直接的制約）ではないため。

 **趣旨** : 儀礼的行為と思想表明の区別。
 **解説** : ピアノ伴奏事件（H19）に続く判例。個人の歴史観によっては「思想への制約」と感じる（間接的制約）としても、公立学校の教職員としての職務命令であり、教育上の行事における慣習的な儀礼行為の要請に過ぎないため、受忍限度内であるとした。
 **条文** : 憲法19条
 **出典** : 君が代起立斉唱拒否事件（最判平23.5.30）
