# Card: 外国人に対して、憲法上「入国の自由」 や 「在留の権利」は保障されるか？

## Question
外国人に対して、憲法上**「入国の自由」 **や** 「在留の権利」**は保障されるか？

## Answer
保障されない。 
（入国・在留は権利ではなく、国家の恩恵的措置あるいは裁量事項とされる。）

 **趣旨** : 国際慣習法上、国家は外国人を自国に受け入れる義務を負わず、受入の可否は国家の主権的行為とされるため。
 **解説** : 憲法22条1項の「居住・移転の自由」は、文言上も「外国へ」の移転（出国）は含むが、外国人の入国まで保障するものではない。入国が権利として保障されない以上、在留の継続（更新）も権利としては保障されない。
 **条文** : 憲法22条1項
