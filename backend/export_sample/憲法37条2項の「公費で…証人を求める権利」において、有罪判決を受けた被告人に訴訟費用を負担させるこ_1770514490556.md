# Card: 憲法37条2項の「公費で…証人を求める権利」において、有罪判決を受けた被告人に訴訟費用を負担させるこ

## Question
憲法37条2項の「公費で…証人を求める権利」において、有罪判決を受けた被告人に訴訟費用を負担させることは許されるか？

## Answer
**許される（合憲）** 。

「公費で」とは、国による **一時立替え** を意味し、最終的な費用免除（無償）までは意味しない。

 **趣旨** : 資力のない被告人でも、十分な防御活動（証人喚問）を行えるようにする「武器対等の原則」の実質化。
 **解説** : 憲法が保障するのは、手続段階で被告人が費用を前払いしなくて済む（貧困により証人を呼べない事態を防ぐ）ことです。事後的に、有罪となった被告人に費用償還を求めることは禁止されていません。
 **条文** : 憲法37条2項
 **出典** : 最大判昭23.12.27
