# Card: 選挙無効訴訟において、選挙自体は違憲・違法だが、選挙の効力は有効とする（請求棄却）判決手法は？

## Question
選挙無効訴訟において、選挙自体は違憲・違法だが、選挙の効力は有効とする（請求棄却）判決手法は？

## Answer
**事情判決の法理**  （行政事件訴訟法31条の準用・流用）

 **趣旨** : 選挙を無効にすることによる政治的空白や社会的混乱（公共の福祉への重大な支障）を回避するため。
 **解説** : 選挙の定数配分が違憲でも、選挙全体を無効にすると全議員が失職し、国会機能が停止してしまう。そこで裁判所は、主文で**「違憲であることを宣言」**しつつ、選挙無効の請求自体は棄却するという手法をとる。
 **条文** : 行政事件訴訟法31条
 **出典** : 最大判昭51.4.14
