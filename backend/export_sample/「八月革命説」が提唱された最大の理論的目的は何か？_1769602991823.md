# Card: 「八月革命説」が提唱された最大の理論的目的は何か？

## Question
「八月革命説」が提唱された最大の理論的目的は何か？

## Answer
明治憲法と日本国憲法の間の「主権の所在の変更」を法的に正当化するため。

 **趣旨** : 明治憲法73条（改正規定）を用いた日本国憲法の制定が、改正の限界（天皇主権の維持）を超えているという矛盾を解消する。 
 **条文** : 憲法制定権力論に基づく。 
 **判例・あてはめ** : ポツダム宣言受諾により「法的革命」が起き、主権が天皇から国民に移転したと擬制する。これにより、新憲法は明治憲法の「改正」ではなく、国民が自ら制定した「別個の憲法」として有効となる。
