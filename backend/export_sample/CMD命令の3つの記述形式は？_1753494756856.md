# Card: CMD命令の3つの記述形式は？

## Question
CMD命令の3つの記述形式は？

## Answer
Exec形式、Shell形式、ENTRYPOINT引数形式<br><pre><code># Exec形式（推奨）
CMD ["python", "app.py"]

# Shell形式
CMD python app.py

# ENTRYPOINT引数形式
ENTRYPOINT ["python"]
CMD ["app.py"]</code></pre>
