# Card: 内在・外在二元的制約説の理論的欠陥として、憲法13条（幸福追求権）との関係で指摘される批判は何か？

## Question
内在・外在二元的制約説の理論的欠陥として、憲法13条（幸福追求権）との関係で指摘される批判は何か？

## Answer
13条を訓示規定とするため、新しい人権（プライバシー権等）を基礎づけることができなくなる。

 **趣旨** : 社会の変化に伴う新しい権利利益の保護に、憲法が適応できなくなるのを防ぐため。
 **解説** : 現代では、プライバシー権や肖像権など、個別の条文にない権利を13条の「幸福追求権」から導き出すのが定石である。しかし、二元説のように13条を「単なる心構え（訓示）」としてしまうと、これらの新しい人権に法的根拠を与えられなくなるという致命的なバグが生じる。
 **条文** : 憲法13条
