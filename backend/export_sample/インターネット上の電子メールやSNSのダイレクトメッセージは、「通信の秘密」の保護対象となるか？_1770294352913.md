# Card: インターネット上の電子メールやSNSのダイレクトメッセージは、「通信の秘密」の保護対象となるか？

## Question
インターネット上の電子メールやSNSのダイレクトメッセージは、「通信の秘密」の保護対象となるか？

## Answer
なる。

郵便や電話に限らず、 **特定人から特定人への情報の伝達** であれば、媒体を問わず保護される。

 **趣旨** : 通信技術の進歩にかかわらず、私的領域における自由な意思疎通を保障するという憲法の目的を達成するため。
 **解説** : 現代においてインターネットは主要な通信手段である。「通信」の定義は「特定の者に対する伝達」であり、不特定多数に向けた「ウェブサイトへの書き込み」等とは区別される。
 **条文** : 憲法21条2項後段
