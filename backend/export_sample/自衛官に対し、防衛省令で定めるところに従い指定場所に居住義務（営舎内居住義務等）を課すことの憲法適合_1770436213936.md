# Card: 自衛官に対し、防衛省令で定めるところに従い指定場所に居住義務（営舎内居住義務等）を課すことの憲法適合

## Question
自衛官に対し、防衛省令で定めるところに従い指定場所に居住義務（営舎内居住義務等）を課すことの憲法適合性は？

## Answer
合憲（公共の福祉による合理的な制限）。

 **趣旨** : 国防・災害派遣等の任務遂行のため、厳格な規律と即応体制を維持する必要があるため。
 **解説** : 居住場所を指定されることは強力な制約だが、自衛隊という組織の任務の公共性・特殊性に鑑みれば、合理的な必要性があると解される（公務員の特別な法律関係に伴う制約）。一般市民に対する規制とは異なる基準で判断される例として記憶する。
 **条文** : 憲法22条1項、自衛隊法55条
