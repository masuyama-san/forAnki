# Card: 明治憲法下の「国家法人説」において、主権（統治権）は誰に帰属すると考えられたか？

## Question
明治憲法下の「国家法人説」において、主権（統治権）は誰に帰属すると考えられたか？

## Answer
国家（法人としての国家）

 **趣旨** : 主権を法人である国家に帰属させ、天皇をその最高機関（機関説）と位置づけることで、君主の権力を法の枠内に置く立憲主義を説明した。 
 **条文** : 大日本帝国憲法第1条・第4条の解釈。 
 **判例・あてはめ** : 当初は学説として広く受け入れられたが、後に「天皇機関説事件」として政治的弾圧の対象となった。
