# Card: 加持祈祷（かじきとう）事件において、宗教行為として行われた暴行が処罰された理由は？

## Question
加持祈祷（かじきとう）事件において、宗教行為として行われた暴行が処罰された理由は？

## Answer
他人の生命・身体に危害を及ぼす行為は、信教の自由の**限界（公共の福祉による制約）**を超えるため。

 **趣旨** : 信仰の自由は絶対だが、外部的行為（宗教的行為）は他者権利との関係で規制を受ける。
 **解説** : 精神障害者の平癒祈願として暴行を加え死亡させた事案。たとえ宗教的信念に基づく行為であっても、刑罰法規に触れ、他人の基本的人権を侵害する場合は免責されない。
 **条文** : 憲法20条1項、13条
 **出典** : 最判昭38.5.15
