# Card: 「検閲（絶対的禁止）」と「事前抑制（原則的禁止）」の処理の違いは？

## Question
「検閲（絶対的禁止）」と「事前抑制（原則的禁止）」の処理の違いは？

## Answer
-  **検閲** : 21条2項の定義（6要件）を満たすもの。 **例外なく違憲** 。
-  **事前抑制** : 検閲には当たらないが、発表前に抑止するもの（例：裁判所の仮処分、税関検査）。 **原則違憲** だが、厳格な要件下で **例外的に合憲** となりうる。

 **趣旨** : 表現が市場に出る前に抑止することは、検閲に近い弊害（情報の隠蔽・萎縮）を生むため、事後規制よりも厳しい基準で審査される。
 **解説** : 「検閲ではない」＝「直ちに合憲」ではない点に注意。検閲でなくとも事前抑制に当たる場合は、事後処罰よりも厳しい基準（明白かつ現在の危険など）で正当化事由が吟味される。
 **条文** : 憲法21条1項、2項
