# Card: 損失補償の「支払時期」について、憲法29条3項は事前の支払（同時履行）を義務付けているか？

## Question
損失補償の「支払時期」について、憲法29条3項は事前の支払（同時履行）を義務付けているか？

## Answer
義務付けていない（事後の支払でも合憲）

 **趣旨** : 公共事業の迅速な遂行を可能にするため、手続的な柔軟性を認める必要がある。
 **解説** : 憲法29条3項は「補償」を求めているが、その支払時期についてまでは規定していない。判例（最大判昭24.7.13）は、補償が財産の供与と交換的に同時に履行されるべきことまでは憲法の保障するところではないとし、相当な期間内の事後払いを定める法律も合憲としている。
 **条文** : 憲法29条3項
