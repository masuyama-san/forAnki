# Card: プライバシー権侵害の不法行為が成立するための3要件（「宴のあと」事件基準）は？

## Question
プライバシー権侵害の不法行為が成立するための3要件（「宴のあと」事件基準）は？

## Answer
-  **私生活上の事実** 、またはそれらしく受け取られる事柄であること（私事性）。
- 一般人の感受性を基準にして、 **公開を欲しない** 事柄であること（秘匿性）。
- 一般の人々に **未だ知られていない** 事柄であること（非公知性）。

 **趣旨** : 表現の自由と個人の私生活の平穏との調和を図るための客観的基準。
 **解説** : 日本で初めてプライバシー権を法的権利として認めたリーディングケース。特に「秘匿性」の判断において、主観的な「本人が嫌がっているか」だけでなく、「一般人の感受性を基準」とする客観説を採用している点が重要。
 **条文** : 民法709条、憲法13条
 **出典** : 東京地判昭39.9.28（「宴のあと」事件）
