# Card: レペタ事件判決における、法廷での「傍聴人のメモ（筆記行為）」の憲法上の位置づけは？

## Question
レペタ事件判決における、法廷での「傍聴人のメモ（筆記行為）」の憲法上の位置づけは？

## Answer
憲法21条1項の **精神に照らして尊重** される。

（権利そのものではないが、正当な理由なく制限できない）

 **趣旨** : 筆記は、見聞した裁判内容を認識・記憶するための補助的行為であり、国民の「知る権利」に資するため。
 **解説** : 判例は、筆記行為自体を憲法上の「権利」とは認めなかったが、「尊重されるべき」とした。したがって、法廷警察権による制限は、公正・円滑な訴訟運営のために真に不可欠な場合に限られる。なお、現在は裁判所規則が改正され、原則自由となっている。
 **条文** : 憲法21条1項、82条
 **出典** : レペタ事件（最大判平元.3.8）
