# Card: 女性の再婚禁止期間違憲判決（平27）において、違憲と判断された期間の範囲は？

## Question
女性の再婚禁止期間違憲判決（平27）において、違憲と判断された期間の範囲は？

## Answer
**100日を超える期間** について違憲

 **趣旨** : 父性の推定の重複を回避するという目的（民法772条）に対し、過剰な制約を避けるため。
 **解説** : 判決当時、計算上「100日」の禁止期間があれば父性の重複は回避可能であった。そのため、それを超える期間（当時は6ヶ月）の禁止は、過剰な制約として違憲とされた。この判決は「期間制限自体」を全否定したわけではない点に注意。
 **条文** : 憲法14条1項、旧民法733条
 **出典** : 最大判平27.12.16
