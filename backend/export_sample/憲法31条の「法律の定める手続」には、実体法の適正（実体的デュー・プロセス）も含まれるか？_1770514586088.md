# Card: 憲法31条の「法律の定める手続」には、実体法の適正（実体的デュー・プロセス）も含まれるか？

## Question
憲法31条の「法律の定める手続」には、実体法の適正（実体的デュー・プロセス）も含まれるか？

## Answer
**含まれる** 。

手続の法定・適正だけでなく、 **実体の法定・適正** も要求する。

 **趣旨** : 形式的に法律で手続を定めさえすれば、どんな不当な刑罰でも許されるとすると、人権保障の機能を果たせないため。
<div style="display: inline !important;"> **解説** : 通説・判例は、31条を英米法のDue Process Clauseと同様に解釈し、以下の4要素を含むと考えます。

- 手続法定主義
- 手続の適正（告知・聴聞）
- 罪刑法定主義（実体の法定）
- 実体の適正（明確性の原則、罪刑の均衡）
 **条文** : 憲法31条
