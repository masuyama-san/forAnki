# Card: docker execで対話モード（インタラクティブ）でコンテナに接続するオプションは？

## Question
docker execで対話モード（インタラクティブ）でコンテナに接続するオプションは？

## Answer
docker exec -it<br><code>-i</code> (--interactive) : 標準入力を開く<br><code>-t</code> (--tty) : 擬似端末を割り当てる<br><pre><code>docker exec -it webapp bash         # webappコンテナでbashシェルを対話実行
docker exec -it database mysql      # データベースに対話的に接続
docker exec -it alpine sh           # Alpine Linuxコンテナでシェル起動</code></pre>
