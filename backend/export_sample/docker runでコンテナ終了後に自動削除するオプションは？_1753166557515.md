# Card: docker runでコンテナ終了後に自動削除するオプションは？

## Question
docker runでコンテナ終了後に自動削除するオプションは？

## Answer
docker run --rm<br>--rm："remove"（削除する）<br><pre><code>docker run --rm ubuntu echo "test"    # 実行後に自動でコンテナ削除
docker run -it --rm alpine sh         # 対話シェル終了後に自動削除
docker run --rm -v $(pwd):/app node npm test  # テスト実行後に自動削除</code></pre>
