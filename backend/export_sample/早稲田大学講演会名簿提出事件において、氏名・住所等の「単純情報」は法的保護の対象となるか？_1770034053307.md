# Card: 早稲田大学講演会名簿提出事件において、氏名・住所等の「単純情報」は法的保護の対象となるか？

## Question
早稲田大学講演会名簿提出事件において、氏名・住所等の「単純情報」は法的保護の対象となるか？

## Answer
**対象となる。**  （単純な識別情報であっても、本人の承諾なくみだりに開示されない自由を有する）

 **趣旨** : 現代社会において、自己の情報をコントロールする権利（または期待）を保護するため。
 **解説** : 秘匿性の低い情報（氏名・学籍番号等）であっても、大学側（私人）が本人の同意なく警察へ渡す行為は、正当な理由がない限りプライバシー権侵害（不法行為）となると判示した。情報の性質による保護の濃淡はあるが、保護対象外ではない点に注意。
 **条文** : 民法709条（憲法13条の趣旨を参酌）
 **出典** : 最判平15.9.12
