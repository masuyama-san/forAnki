# Card: docker runで環境変数を設定するオプションは？

## Question
docker runで環境変数を設定するオプションは？

## Answer
docker run -e 変数名=値<br>-e (--env): "environment"（環境変数）<br><pre><code>docker run -e NODE_ENV=production node     # NODE_ENV環境変数を設定
docker run -e MYSQL_ROOT_PASSWORD=secret mysql  # MySQL root パスワード設定
docker run -e DB_HOST=localhost -e DB_PORT=5432 myapp  # 複数環境変数</code></pre>
