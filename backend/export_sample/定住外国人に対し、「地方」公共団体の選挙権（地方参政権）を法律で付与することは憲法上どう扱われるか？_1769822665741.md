# Card: 定住外国人に対し、「地方」公共団体の選挙権（地方参政権）を法律で付与することは憲法上どう扱われるか？

## Question
定住外国人に対し、**「地方」**公共団体の選挙権（地方参政権）を法律で付与することは憲法上どう扱われるか？

## Answer
憲法上の権利としては保障されないが、法律で付与することは憲法上**「禁止」されていない**（許容説）。

 **趣旨** : 「国民主権原理（原則）」と「住民自治（地方自治の特殊性）」の調和。
 **解説** : 憲法93条2項の「住民」とは日本国民を指し、外国人に選挙権は保障されない（権利説の否定）。しかし、永住者等は地域社会と密接な関係を持つため、法律で地方選挙権を付与する措置を講ずることは、立法政策として許容される。
 **条文** : 憲法15条1項、93条2項
