# Card: 大学教授が「授業の内容」について大学当局から制限を受けた場合、憲法23条の問題となるか？

## Question
大学教授が「授業の内容」について大学当局から制限を受けた場合、憲法23条の問題となるか？

## Answer
**問題となる（保障の及ぶ範囲）。** 

教授の「学問の自由」と、大学の「教育課程編成の自治」が衝突する場面である。

 **趣旨** : 大学教授の授業は、研究発表の側面を持つため、普通教育（小中高）の教師よりも広く自由が保障される。
 **解説** : 大学の自治は通常、外部に対する防御権だが、内部において教授個人の研究・教育活動を不当に制約する場合、教授は「学問の自由」を根拠に対抗しうる。
 **条文** : 憲法23条
