# Card: 塩見訴訟において、在日外国人への国民健康保険の適用除外（条例）は合憲とされたか？

## Question
塩見訴訟において、在日外国人への国民健康保険の適用除外（条例）は合憲とされたか？

## Answer
**合憲**  （外国人をどの範囲で保護するかは立法裁量に委ねられる）

 **趣旨** : 限られた財源の中で、自国民の保護を優先することは許容される。
 **解説** : 憲法25条の社会保障は、外国人に対しては権利として保障されず、恩恵的な措置にとどまる。したがって、条例で外国人を適用の対象とするか否かは、地方公共団体の **政治的・政策的裁量** に委ねられ、適用しないことも違憲ではないとした。
 **条文** : 憲法14条1項、25条
 **出典** : 最判平元.3.2
