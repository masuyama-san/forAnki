# Card: 海外渡航の自由の制約に関して、判例（帆足計事件）が採用した合憲性判定基準は何か？

## Question
海外渡航の自由の制約に関して、判例（帆足計事件）が採用した合憲性判定基準は何か？

## Answer
**合理性の基準** （公共の福祉のためにする合理的な制限）。

 **趣旨** : 外交政策や国際情勢判断には、司法よりも行政府（外務大臣）の広範な裁量が尊重されるべき領域であるため。
 **解説** : 裁判所は、「著しくかつ直接に日本国の利益または公安を害する行為を行うおそれ」がある場合に旅券の発給を拒否できるとした旅券法の規定を合憲とした。精神的自由の側面があるにもかかわらず、厳格な基準（LRA等）ではなく、比較的緩やかな「合理的制限」という言葉を用いている点が特徴的である。
 **条文** : 憲法22条2項、旅券法13条1項
 **出典** : 最大判昭33.9.10
