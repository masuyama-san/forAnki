# Card: 財産権侵害について「特別の犠牲」があるにもかかわらず、根拠法に損失補償の規定がない場合、判例（河川付

## Question
財産権侵害について「特別の犠牲」があるにもかかわらず、根拠法に損失補償の規定がない場合、判例（河川付近地制限令事件）はどのような救済を認めたか？

## Answer
憲法29条3項を直接の根拠として、補償請求を認める（直接請求権説）

 **趣旨** : 法律自体を違憲無効にして公益目的（河川管理等）を阻害するよりも、補償請求権を認めて救済する方が、合憲性を維持しつつ公平を図れるからである。
 **解説** : 判例（最大判昭43.11.27）は、法律に補償規定が欠けている場合でも、その法律を直ちに違憲無効とするのではなく、憲法29条3項を直接の根拠として、国に対して損失補償を請求する余地があるとした。これにより、法の効力を維持したまま個人の救済が可能となる。
 **条文** : 憲法29条3項
