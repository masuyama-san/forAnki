# Card: 一元的外在制約説とはどのような見解か？

## Question
一元的外在制約説とはどのような見解か？

## Answer
人権の「外」にある一般的原理（公益）が、人権を制約するとする見解。

 **趣旨** : 国家目的（公益）のために、広範な人権制約を正当化しようとする初期の理論。
 **解説** : この説は「公共の福祉＝公益」という外的なハンマーで、人権という壁を叩くイメージ。12条・13条を「あらゆる人権を制約できる万能の根拠（一般原理）」として使うため、権利の性質に関わらず一律に制約を肯定しやすい。
 **条文** : 憲法12条、13条
