# Card: 税関検査（輸入禁制品の審査）は、「検閲」に当たるか？

## Question
税関検査（輸入禁制品の審査）は、「検閲」に当たるか？

## Answer
**当たらない** 。

- 輸入禁止であっても、 **国外ですでに発表済み** の表現物であり、発表そのものの禁止ではない。
- 司法審査の機会が与えられている。

 **趣旨** : 関税徴収手続きの一環であり、行政権による思想内容の審査・規制を目的としていない。
 **解説** : 「発表前の審査」に見えるが、国外発表済みである点などから検閲には当たらないとした。ただし、実質的な「事前抑制」には当たるため、明確性の原則などの観点から厳格に審査された。
 **条文** : 憲法21条2項
 **出典** : 最大判昭59.12.12（税関検査事件）
