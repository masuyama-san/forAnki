# Card: 日本国憲法下において、伝統的な「特別権力関係論（法治主義の排除・人権の包括的制限）」は認められるか？

## Question
日本国憲法下において、伝統的な「特別権力関係論（法治主義の排除・人権の包括的制限）」は認められるか？

## Answer
**認められない（全面的に否定）**  
※現在は、公務員や在監者等の関係も「法の支配」に服し、人権制限には個別の法的根拠が必要とされる。

 **趣旨** : すべての国民は基本的人権を享有し（憲法11条）、法の下に平等である（14条）ため、法から自由な聖域は許されない。
 **解説** : 明治憲法下の「公法上の特別の原因」に基づく理論は完全に否定された。現在は、公務員や在監者にも原則として人権が及ぶことを前提としつつ、公共の福祉（13条）による「内在的制約」として、その地位や職務の性質に応じた「必要最小限度」等の合理的制限が許容される（修正された特別権力関係論とも呼ばれる実質的判断）。
 **条文** : 憲法11条、13条
