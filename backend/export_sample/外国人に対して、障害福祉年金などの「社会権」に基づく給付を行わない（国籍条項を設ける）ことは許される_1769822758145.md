# Card: 外国人に対して、障害福祉年金などの「社会権」に基づく給付を行わない（国籍条項を設ける）ことは許される

## Question
外国人に対して、障害福祉年金などの**「社会権」**に基づく給付を行わない（国籍条項を設ける）ことは許されるか？

## Answer
許される（合憲）。

 **趣旨** : 限られた財源の配分に関する立法府の広範な裁量の尊重。
 **解説** : 社会権（25条）は抽象的な権利であり、その具体化は国の財政事情に依存する。したがって、対象設定には立法府に広い裁量があり、自国民を在留外国人より優先的に扱うことも、合理的な区別として許容される。
 **条文** : 憲法25条、14条1項
