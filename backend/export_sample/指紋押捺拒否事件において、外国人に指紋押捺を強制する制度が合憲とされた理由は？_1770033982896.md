# Card: 指紋押捺拒否事件において、外国人に指紋押捺を強制する制度が合憲とされた理由は？

## Question
指紋押捺拒否事件において、外国人に指紋押捺を強制する制度が合憲とされた理由は？

## Answer
指紋は個人の **内心・思想に関わらない識別情報** であり、制度目的（公正な管理）も合理的で、手段も **必要かつ相当** であるため（受忍限度内）。

 **趣旨** : 国家による個人情報の強制採取が許容される「正当な理由」の限界を示した。
 **解説** : 「みだりに指紋押捺を強制されない自由」は13条で保障されるが、指紋自体には思想性がないため保護の程度は相対的に低い。代替手段の乏しさや行政目的の重要性から、公共の福祉による合理的制約として許容された。
 **条文** : 憲法13条
 **出典** : 最判平7.12.15
