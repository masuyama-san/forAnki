# Card: 国籍法違憲判決（婚外子の国籍取得）において、違憲とされた要件は何か？

## Question
国籍法違憲判決（婚外子の国籍取得）において、違憲とされた要件は何か？

## Answer
**準正要件（父母の婚姻）**  （認知に加え、父母の婚姻を国籍取得要件とすることは過剰）

 **趣旨** : 子に責任のない事由により、国籍という重要な権利を差別してはならない。
 **解説** : 日本人父から認知されていれば「日本との密接な結びつき」は認められる。それに加えて「父母の婚姻」まで要求することは、立法目的との間で **合理的関連性を欠く過剰な手段** であり、違憲である。
 **条文** : 憲法14条1項、旧国籍法3条1項
 **出典** : 最大判平20.6.4
