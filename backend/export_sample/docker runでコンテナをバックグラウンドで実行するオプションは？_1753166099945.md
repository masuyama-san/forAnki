# Card: docker runでコンテナをバックグラウンドで実行するオプションは？

## Question
docker runでコンテナをバックグラウンドで実行するオプションは？

## Answer
docker run -d<br>-d (--detach)："detached"（デタッチモードで起動）<br><pre><code>docker run -d nginx           # nginxをバックグラウンドで実行
docker run -d --name webserver nginx  # 名前付きでバックグラウンド実行
docker run -d -p 80:80 nginx  # ポート指定でバックグラウンド実行</code></pre>
