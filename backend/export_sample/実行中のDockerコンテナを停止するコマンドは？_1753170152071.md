# Card: 実行中のDockerコンテナを停止するコマンドは？

## Question
実行中のDockerコンテナを停止するコマンドは？

## Answer
docker stop<br><pre><code>docker stop my-container        # my-containerという名前のコンテナを停止
docker stop c1d2e3f4             # コンテナIDで停止
docker stop web-server database  # 複数のコンテナを同時停止</code></pre>
