# Card: Dockerイメージの作成履歴（レイヤー情報）を表示するコマンドは？

## Question
Dockerイメージの作成履歴（レイヤー情報）を表示するコマンドは？

## Answer
docker history<br><pre><code>docker history nginx                # nginxイメージの履歴を表示
docker history my-app:latest        # 自作イメージの履歴を表示
docker history ubuntu:20.04         # Ubuntu 20.04の履歴を表示</code></pre>
