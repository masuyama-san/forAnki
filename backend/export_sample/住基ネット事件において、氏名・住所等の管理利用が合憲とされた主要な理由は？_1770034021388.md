# Card: 住基ネット事件において、氏名・住所等の管理利用が合憲とされた主要な理由は？

## Question
住基ネット事件において、氏名・住所等の管理利用が合憲とされた主要な理由は？

## Answer
- 管理される情報は個人の識別情報であり、 **秘匿性が低い** 。
- システム技術上の不備による **漏洩の具体的危険性が認められない** 。

 **趣旨** : 行政の効率化（公共の福祉）と、自己情報コントロール権の調和。
 **解説** : 最高裁は、氏名等の単純情報も「みだりに第三者に開示されない自由」として13条で保障するとした。しかし、これらは他者との識別情報であり内面の秘密ではないため、システム上の漏洩防止措置がある限り、具体的危険はなく合憲とした。
 **条文** : 憲法13条
 **出典** : 最判平20.3.6
