# Card: Dockerfileでベースイメージを指定する命令は？

## Question
Dockerfileでベースイメージを指定する命令は？

## Answer
FROM<br><pre><code>FROM ubuntu:20.04              # Ubuntu 20.04をベースイメージに
FROM node:16-alpine            # Node.js 16のAlpine版をベースに
FROM nginx:latest              # 最新のnginxイメージをベースに</code></pre>
