# Card: TBSビデオテープ事件において、 未放映部分を含む テープ（マザーテープ）の差押えが合憲とされた理由

## Question
TBSビデオテープ事件において、 **未放映部分を含む** テープ（マザーテープ）の差押えが合憲とされた理由は？

## Answer
編集・放映自体は完了しており、将来の取材の自由への影響（萎縮効果）にとどまり、報道そのものが不可能になるわけではないため。

 **趣旨** : 未放映素材は取材源の秘密に近く保護の必要性が高いが、重大事件の捜査の必要性が優越する場合がある。
 **解説** : 博多駅事件（放映済み）と異なり、未放映部分が含まれていたが、最高裁は「編集も放映も済んでいる」点に着目。マザーテープを押収しても、既に報じた内容が覆るわけではなく、不利益は「将来の取材協力が得られなくなる」という一般的・抽象的なおそれにすぎないとして、公正な刑事裁判の要請を優先させた。
 **条文** : 憲法21条、刑事訴訟法99条
 **出典** : TBSビデオテープ差押事件（最決平2.7.9）
