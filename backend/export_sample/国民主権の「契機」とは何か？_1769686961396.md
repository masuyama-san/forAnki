# Card: 国民主権の「契機」とは何か？

## Question
国民主権の「契機」とは何か？

## Answer
- 正統性の契機
- 権力性の契機（権力的契機）

 **趣旨** : 国民主権原理と象徴天皇制を論理的に整合させ、国民による政治参加の根拠を明確にするため。 
 **解説** : 「正統性の契機」とは、国家権力行使を正当化する究極の根拠が国民に存するという原理であり、天皇の地位の根拠（1条）となる。一方、「権力性の契機」とは、国民が自ら国政決定の権力を行使する実力的な側面を指し、憲法改正（96条）や選挙権の行使に現れる。象徴天皇制は、天皇に「権力」を認めず「正統性」の根拠として国民と結びつけることで、国民主権と両立している。 
 **条文** : 憲法1条、前文1項
