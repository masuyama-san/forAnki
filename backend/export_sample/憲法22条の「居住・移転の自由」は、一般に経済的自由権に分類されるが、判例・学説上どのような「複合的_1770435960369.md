# Card: 憲法22条の「居住・移転の自由」は、一般に経済的自由権に分類されるが、判例・学説上どのような「複合的

## Question
憲法22条の「居住・移転の自由」は、一般に経済的自由権に分類されるが、判例・学説上どのような「複合的性質」を有すると解されているか？

## Answer
-  **経済的自由** の側面
-  **精神的自由** の側面
-  **人身の自由** の側面

 **趣旨** : 移動の自由は、単なる経済活動の基盤（職業選択との結合）にとどまらず、個人の知見拡大や身体的安全にも関わる基本的人権の根幹であるため。
<div style="display: inline !important;"> **解説** :

-  **経済的自由** : 封建的な土地緊縛からの解放。職業選択の自由と密接に関連する。
-  **精神的自由** : 学術・技芸の習得、取材や留学で見聞を広めることによる人格形成の基盤。
-  **人身の自由** : 不当な身体拘束や強制収容からの解放。
※論文試験では、当該規制がどの側面（特に精神・人身）に強く及ぶかを指摘し、それに応じて違憲審査基準の厳格度を調整する論述が求められる。
 **条文** : 憲法22条
