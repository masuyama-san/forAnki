# Card: 外国人による「指紋押捺拒否」は、憲法上の権利として認められるか？

## Question
外国人による**「指紋押捺拒否」**は、憲法上の権利として認められるか？

## Answer
認められない。 
（指紋押捺制度自体は、必要かつ合理的なものであり、憲法13条に違反しない。）

 **趣旨** : 在留外国人の公正な管理という行政目的の達成のため。
 **解説** : 指紋押捺制度は、個人の私生活の自由を一定程度制約するものの、戸籍のない外国人の人物特定のために必要かつ合理的な制度とされた（※現在は制度変更済だが、判例知識として重要）。外国人の権利も「公共の福祉」による制約を受ける一例。
 **条文** : 憲法13条
