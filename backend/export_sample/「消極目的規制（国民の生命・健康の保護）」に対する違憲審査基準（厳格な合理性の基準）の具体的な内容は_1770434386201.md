# Card: 「消極目的規制（国民の生命・健康の保護）」に対する違憲審査基準（厳格な合理性の基準）の具体的な内容は

## Question
「消極目的規制（国民の生命・健康の保護）」に対する違憲審査基準（厳格な合理性の基準）の具体的な内容は？

## Answer
重要な公共の利益のために必要かつ合理的であり、かつ、より緩やかな規制手段では目的を達成できないこと（LRAの基準）。

 **趣旨** : 自由を奪う強力な規制は、他に手段がない場合にのみ許されるべきという厳格な要請。
 **解説** : 消極目的規制（例：薬事法）においては、①目的が「重要」であること、②手段が目的達成のために「実質的関連性」を持ち、かつ「必要最小限度（Less Restrictive Alternativeが存在しない）」であることが要求される。
 **条文** : 憲法第22条第1項
 **出典** : 最大判昭50.4.30（薬局距離制限事件）
