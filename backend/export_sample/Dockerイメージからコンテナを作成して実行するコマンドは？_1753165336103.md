# Card: Dockerイメージからコンテナを作成して実行するコマンドは？

## Question
Dockerイメージからコンテナを作成して実行するコマンドは？

## Answer
docker run<br><pre><code>docker run nginx              # nginxコンテナを実行
docker run ubuntu echo "Hello"  # ubuntuでechoコマンドを実行
docker run -it python bash   # Pythonコンテナでbashを対話実行</code></pre>
