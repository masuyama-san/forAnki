# Card: 第三者所有物没収事件において、被告人は「第三者の権利侵害（手続不備）」を理由に上告できるか？

## Question
第三者所有物没収事件において、被告人は「第三者の権利侵害（手続不備）」を理由に上告できるか？

## Answer
**できる** 。

（被告人自身に具体的な不利益リスクがあるため例外的に肯定）

 **趣旨** : 原則として他人の権利侵害を援用することはできないが（第三者の予備的主張の禁止）、被告人自身が実質的な被害を受ける場合は救済する必要がある。
 **解説** : 本件では、没収執行不能により、所有権を剥奪されなかった第三者から被告人が**「求償（損害賠償請求）」を受ける危険**があります。この具体的利害関係を根拠に、例外的に被告人の上告理由適格が認められました。
 **条文** : 刑事訴訟法405条、憲法31条
 **出典** : 最大判昭37.11.28（第三者所有物没収事件）
