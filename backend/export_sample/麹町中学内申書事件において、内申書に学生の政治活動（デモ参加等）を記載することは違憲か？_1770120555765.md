# Card: 麹町中学内申書事件において、内申書に学生の政治活動（デモ参加等）を記載することは違憲か？

## Question
麹町中学内申書事件において、内申書に学生の政治活動（デモ参加等）を記載することは違憲か？

## Answer
**合憲（違反しない）。** 

記載されたのは「外部的行為（活動実績）」の事実であり、生徒の思想・信条そのものではないため。

 **趣旨** : 教育評価資料としての正確性と、内面の自由の保護の調整。
 **解説** : 学校長には調査書の作成権限がある。内申書の記載は、高校入試の資料とするための客観的な事実報告に過ぎず、生徒の思想そのものを調査したり、それによって不利益に取り扱ったわけではない。「外部的行為」と「内心」を区別する重要判例。
 **条文** : 憲法19条
 **出典** : 麹町中学内申書事件（最判昭63.7.15）
