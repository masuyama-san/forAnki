# Card: 判例（帆足計事件）において、憲法22条2項の「外国に移住する自由（永住）」の文言から、一時的な「海外

## Question
判例（帆足計事件）において、憲法22条2項の「外国に移住する自由（永住）」の文言から、一時的な「海外旅行の自由」も保障されるか？

## Answer
保障される
「大は小を兼ねる」論理

 **趣旨** : より強力な権利（永住・国籍離脱）が認められるなら、それに包含される軽度の権利（一時旅行）も当然に認められるべきという解釈手法。
 **解説** : 「移住（永住）」という、日本国の支配を離脱し国籍離脱にも通ずる強力な権利が保障されている以上、それよりも程度の低い「一時的な旅行」も当然に同条項で保障されると解釈するのが合理的である。憲法が永住のための出国を保障しながら、旅行のための出国を認めないと解するのは不合理だからである。
 **条文** : 憲法22条2項
 **出典** : 最大判昭33.9.10
