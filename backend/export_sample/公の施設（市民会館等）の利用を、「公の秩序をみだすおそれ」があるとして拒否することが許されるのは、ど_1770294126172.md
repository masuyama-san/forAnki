# Card: 公の施設（市民会館等）の利用を、「公の秩序をみだすおそれ」があるとして拒否することが許されるのは、ど

## Question
公の施設（市民会館等）の利用を、「公の秩序をみだすおそれ」があるとして拒否することが許されるのは、どのような場合に限られるか？（泉佐野市民会館事件の規範）

## Answer
単に危険な事態を生ずる蓋然性があるだけでは足りず、 **明らかな差し迫った危険の発生が具体的に予見される場合** に限られる。

 **趣旨** : 「公の施設」は住民の表現活動の場（パブリック・フォーラム）として重要であり、行政による安易な規制（萎縮効果）を防ぐため。
 **解説** : 「公の秩序」という抽象的な要件を厳格に限定解釈した判例。主観的な不安ではなく、客観的事実に基づき、警察の警備等によっても防止できないほどの具体的な危険が迫っていることを要件とする（「明白かつ現在の危険」の法理の影響）。
 **条文** : 憲法21条1項、地方自治法244条
 **出典** : 最判平7.3.7（泉佐野市民会館事件）
