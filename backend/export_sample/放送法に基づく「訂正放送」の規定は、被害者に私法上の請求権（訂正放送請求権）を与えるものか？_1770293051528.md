# Card: 放送法に基づく「訂正放送」の規定は、被害者に私法上の請求権（訂正放送請求権）を与えるものか？

## Question
放送法に基づく「訂正放送」の規定は、被害者に私法上の請求権（訂正放送請求権）を与えるものか？

## Answer
与えない。

（放送事業者の自律的な業務としての公法上の義務にすぎない）

 **趣旨** : 放送内容への介入を最小限にし、放送事業者の自律性（編集の自由）を尊重するため。
 **解説** : 放送法は、真実でない放送がなされた場合に訂正放送を行う義務を定めているが、最高裁はこれを「放送事業者が自律的に行う公法上の義務」と解釈した。被害者が民事訴訟で直接「訂正放送しろ」と請求する根拠にはならない（別途、民法上の不法行為に基づく謝罪広告等は請求しうる）。
 **条文** : 放送法4条、9条
 **出典** : 最判平16.11.25
