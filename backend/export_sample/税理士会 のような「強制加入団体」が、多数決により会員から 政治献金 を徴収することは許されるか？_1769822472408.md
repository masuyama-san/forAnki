# Card: 税理士会 のような「強制加入団体」が、多数決により会員から 政治献金 を徴収することは許されるか？

## Question
**税理士会** のような「強制加入団体」が、多数決により会員から **政治献金** を徴収することは許されるか？

## Answer
許されない（無効）。

 **趣旨** : 実質的な「脱退の自由」がない団体において、少数派会員の「思想・信条の自由（特に消極的自由）」を保護するため。
 **解説** : 税理士会等は法令により加入が義務付けられた公的な団体である。このような団体が、多数決で特定の政党への資金拠出を強制することは、会員に対し「意に反する政治的協力」を強いるものであり、協力義務の限界（目的の範囲）を逸脱し無効である。
 **条文** : 憲法19条、民法34条
