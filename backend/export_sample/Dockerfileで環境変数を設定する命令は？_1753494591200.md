# Card: Dockerfileで環境変数を設定する命令は？

## Question
Dockerfileで環境変数を設定する命令は？

## Answer
ENV<br><pre><code>ENV NODE_ENV=production            # Node.js環境を本番に設定

ENV DATABASE_URL=postgres://localhost:5432/mydb  # データベース接続URL</code></pre>
