# Card: docker logsでリアルタイムでログを監視するオプションは？

## Question
docker logsでリアルタイムでログを監視するオプションは？

## Answer
docker logs -f<br><code>-f </code>: "follow"（追跡）<br><pre><code>docker logs -f webapp           # webappのログをリアルタイム監視
docker logs --follow my-app     # リアルタイムでログを表示
docker logs -f --tail 100 nginx # 最新100行からリアルタイム監視</code></pre>
