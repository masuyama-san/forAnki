# Card: 表現の自由が有する「2つの核心的価値」とは？

## Question
表現の自由が有する「2つの核心的価値」とは？

## Answer
-  **自己実現の価値** （個人の人格的発展）
-  **自己統治の価値** （民主主義への寄与）

 **趣旨** : なぜ表現の自由が、経済的自由など他の人権よりも手厚く保護されるのか（優越的地位）を基礎づける理論。
<div style="display: inline !important;"> **解説** : 

-  **自己実現** : 言いたいことを言い、知りたいことを知る喜び（個人的価値）。
-  **自己統治** : 国民が情報を得て議論し、政治に参加する（社会的価値）。
特に「自己統治」の価値は、取材・報道の自由を強力に保護する論拠となる。
 **条文** : 憲法21条、13条
