# Card: 裁判官による「積極的な政治運動」の禁止（ 寺西判事補事件 ）は、一般職公務員への規制と比較してどのよ

## Question
裁判官による「積極的な政治運動」の禁止（ **寺西判事補事件** ）は、一般職公務員への規制と比較してどのような特徴があるか？

## Answer
一般職公務員よりも **厳格な規制（一律禁止）が許容される** 。 
※「実質的おそれ」の限定解釈を用いず、形式的に該当すれば違法とされる。

 **趣旨** : 裁判官は自ら司法権を行使する主体であり、中立・公正の「実体」だけでなく**「外観（国民からの信頼）」**も維持する必要性が極めて高いため。
 **解説** : 裁判官が特定の政治的集会で積極的に発言することは、それだけで裁判の公正さに疑念を抱かせる。そのため、一般職公務員（堀越基準）のような「実質的危険」の有無にかかわらず、一律に禁止することも憲法21条に反しない。
 **条文** : 裁判所法52条1号、憲法76条
