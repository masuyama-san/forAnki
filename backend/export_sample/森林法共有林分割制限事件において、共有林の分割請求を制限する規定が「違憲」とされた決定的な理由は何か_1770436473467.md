# Card: 森林法共有林分割制限事件において、共有林の分割請求を制限する規定が「違憲」とされた決定的な理由は何か

## Question
森林法共有林分割制限事件において、共有林の分割請求を制限する規定が「違憲」とされた決定的な理由は何か？

## Answer
規制手段が必要性・合理性を欠いていたから（手段の不適合）

 **趣旨** : 立法目的が正当であっても、その手段が過剰で権利侵害が甚だしければ違憲となる（比例原則の実質化）。
 **解説** : 裁判所は、森林経営の安定という「立法目的」自体は正当（公共の福祉に適合）と認めた。しかし、分割を一切禁止しなくとも他の方法で目的達成が可能であること、持分の処分まで困難になること等から、手段として「必要性および合理性を肯定することはできない」と断じ、立法府の裁量逸脱として違憲無効とした。
 **条文** : 憲法29条2項
