# Card: 議員定数不均衡訴訟において、選挙が「違憲」と判断されるための2つの要件（プロセス）は？

## Question
議員定数不均衡訴訟において、選挙が「違憲」と判断されるための2つの要件（プロセス）は？

## Answer
- 投票価値の不均衡が国会の裁量権の限界を超えていること（ **違憲状態** ）。
- 是正のために通常必要な **合理的期間** を経過してもなお、是正されていないこと。

 **趣旨** : 人口変動による格差発生は不可避であり、国会に一定の是正猶予期間（タイムラグ）を与えるため。
 **解説** : 単に格差が大きいだけでは「違憲状態」にとどまる。国会が是正義務を怠り、合理的期間を徒過したと評価されて初めて「違憲」の判断が下される。
 **条文** : 憲法14条1項
 **出典** : 最大判昭51.4.14 等
