# Card: 憲法22条1項の「職業選択の自由」には、職業を選び決定する自由（狭義）だけでなく、何が含まれると解さ

## Question
憲法22条1項の「職業選択の自由」には、職業を選び決定する自由（狭義）だけでなく、何が含まれると解されているか？

## Answer
職業遂行の自由（営業の自由）

 **趣旨** : 職業は、選択した後にその業務を遂行して初めて社会的機能を果たし、自己の人格的価値を全うできるため。
 **解説** : 22条1項の文言は「選択」のみだが、判例はこれに密接不可分な「営業の自由（職業遂行の自由）」も保障に含まれると解している。ただし、営業の自由は社会的相互関連性が強いため、選択の自由そのものに比べて公権力による規制が許容される度合いが高いとされる。
 **条文** : 憲法第22条第1項
 **出典** : 最大判昭47.11.22（小売市場距離制限事件）
