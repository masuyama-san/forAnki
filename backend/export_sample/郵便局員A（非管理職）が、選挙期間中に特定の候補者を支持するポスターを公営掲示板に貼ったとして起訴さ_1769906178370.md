# Card: 郵便局員A（非管理職）が、選挙期間中に特定の候補者を支持するポスターを公営掲示板に貼ったとして起訴さ

## Question
郵便局員A（非管理職）が、選挙期間中に特定の候補者を支持するポスターを公営掲示板に貼ったとして起訴された。 現在の判例理論（堀越基準）を前提とした場合、Aの有罪・無罪を分けるポイントは？

## Answer
Aの行為が、職務権限や勤務実態に照らして**「行政の中立性を損なう実質的なおそれ」**があるか否か。 
（単にポスターを貼った事実だけでは有罪とはならない）

 **趣旨** : 猿払事件（同種の事案で有罪）と堀越事件（実質的危険を要求）の整合性を問う。
 **解説** : 猿払事件当時は一律有罪とされたが、現在の堀越基準では、①管理職でなく、②職務権限も限定的で、③勤務時間外の行為であれば、実質的危険がないとして **無罪** となる可能性が高い。事案が猿払事件と同じでも、適用される法解釈（堀越基準）が変わっている点に注意する。
 **条文** : 国家公務員法102条
