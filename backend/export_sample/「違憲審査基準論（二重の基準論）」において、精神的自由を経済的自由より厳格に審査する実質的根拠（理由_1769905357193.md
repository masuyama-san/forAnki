# Card: 「違憲審査基準論（二重の基準論）」において、精神的自由を経済的自由より厳格に審査する実質的根拠（理由

## Question
「違憲審査基準論（二重の基準論）」において、精神的自由を経済的自由より厳格に審査する実質的根拠（理由）は何か？

## Answer
- 民主政の過程における回復困難性（自己回復能力の欠如）。
- 個人の人格形成における重要性。

 **趣旨** : なぜ一律の「調整」ではなく、権利によって「えこひいき（審査密度の差）」をするのかという理由付け。
 **解説** : 経済的自由への不当な規制は議会（選挙）で是正可能だが、表現の自由が規制されると選挙（民主主義プロセス）自体が機能しなくなるため、裁判所が厳しくチェックする必要がある。この論理により、内在説の枠内で柔軟な審査が可能となる。
 **出典** : 最大判昭47.11.22（小売市場事件）等
