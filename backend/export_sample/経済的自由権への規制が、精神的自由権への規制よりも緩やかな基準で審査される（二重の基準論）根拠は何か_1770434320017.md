# Card: 経済的自由権への規制が、精神的自由権への規制よりも緩やかな基準で審査される（二重の基準論）根拠は何か

## Question
経済的自由権への規制が、精神的自由権への規制よりも緩やかな基準で審査される（二重の基準論）根拠は何か？

## Answer
- 職業活動の持つ「社会的相互関連性」
- 立法府の政策的・技術的な裁量の尊重

 **趣旨** : 経済活動は他者への影響が大きく、かつその調整には高度な政策判断を要するため、司法は抑制的であるべきという考え方。
 **解説** : 精神的自由とは異なり、経済活動は社会生活において他者と密接に関わり合う（社会的相互関連性）。また、社会経済政策の当否は、民主的基盤と調査能力を持つ立法府の判断に委ねるのが適当であり、裁判所の能力を超える場合が多い（司法の限界）。
 **条文** : 憲法第22条第1項
 **出典** : 最大判昭47.11.22
