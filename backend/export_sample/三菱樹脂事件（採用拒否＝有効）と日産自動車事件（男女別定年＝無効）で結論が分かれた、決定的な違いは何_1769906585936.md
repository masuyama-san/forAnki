# Card: 三菱樹脂事件（採用拒否＝有効）と日産自動車事件（男女別定年＝無効）で結論が分かれた、決定的な違いは何

## Question
三菱樹脂事件（採用拒否＝有効）と日産自動車事件（男女別定年＝無効）で結論が分かれた、決定的な違いは何か？

## Answer
**契約締結「前」か、契約締結「後」か** 。 
（採用の自由 vs 労働条件の平等）

 **趣旨** : 段階（フェーズ）に応じた「企業の自由」と「労働者の保護」の利益衡量の変化。
<div style="display: inline !important;"> **解説** :

-  **採用段階** : 企業側の「誰と契約するか選ぶ自由（採用の自由）」が最大限尊重される。
-  **採用後** : 労働者は企業の支配下に置かれるため、不合理な差別からの「保護の必要性」が高まり、公序良俗による修正（介入）が強く働く。
 **条文** : 民法90条
