# Card: docker psですべてのコンテナ（停止中も含む）を表示するオプションは？

## Question
docker psですべてのコンテナ（停止中も含む）を表示するオプションは？

## Answer
docker ps -a<br><code>-a(--all)&nbsp;</code>: "all"（すべて）<br><pre><code>docker ps -a                    # 実行中・停止中すべてのコンテナを表示
docker ps --all                 # 同上（--allの省略形が-a）</code></pre>
