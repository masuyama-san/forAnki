# Card: 外国人の人権享有主体性に関する「マクリーン法理」（在留制度の枠内説）とは何か？

## Question
外国人の人権享有主体性に関する**「マクリーン法理」**（在留制度の枠内説）とは何か？

## Answer
外国人の人権保障は、外国人在留制度の枠内で与えられているにすぎない。

 **趣旨** : 国際法上の国家主権（外国人の受入裁量）を優先し、在留外国人の権利を一定程度制約するため。
 **解説** : 外国人に入国の自由（在留の権利）は保障されず、在留の許否は法務大臣の広範な裁量に委ねられる。したがって、在留中に憲法上保障される政治活動を行ったとしても、更新審査でそれを消極的事情として評価（不利益処分）することは、裁量の範囲内であり合憲とされる。
 **条文** : 憲法22条1項（反対解釈）
