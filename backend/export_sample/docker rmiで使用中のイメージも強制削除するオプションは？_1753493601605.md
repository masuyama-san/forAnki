# Card: docker rmiで使用中のイメージも強制削除するオプションは？

## Question
docker rmiで使用中のイメージも強制削除するオプションは？

## Answer
docker rmi -f<br><code>-f (--force)</code>:&nbsp;"force"（強制）<br><pre><code>docker rmi -f nginx            # nginxを強制削除（コンテナが使用中でも）
docker rmi --force my-app:v1.0 # 強制削除
docker rmi -f $(docker images -q)  # すべてのイメージを強制削除</code></pre>
