# Card: 大学の自治（23条）における「警察権との関係」について、東大ポポロ事件の規範は？

## Question
大学の自治（23条）における「警察権との関係」について、東大ポポロ事件の規範は？

## Answer
大学の **学問の自由と自治を享有しない** 活動（実社会の政治的活動等）には、大学の自治は及ばない。
したがって、そのような集会への警察官の立ち入りは **違憲ではない** 。

 **趣旨** : 「大学の自治」はあくまで「学問的活動」を保護するためのものであり、大学を政治活動の聖域（アジール）にするものではない。
 **解説** : 学生劇団「ポポロ」による演劇発表会への警察官立ち入りが問題となった事例。最高裁は、この集会が「真に学問的な研究・発表」ではなく「実社会の政治的社会的活動」であると認定し、警察権の行使を合憲とした。逆に言えば、純粋な学問的活動に対しては警察権も抑制されるべきことを示唆している。
 **条文** : 憲法23条
 **出典** : 東大ポポロ事件（最大判昭38.5.22）
