# Card: 未成年者 の人権制約（飲酒・喫煙禁止や選挙権制限など）において、成人とは異なる規制が許容される根拠は

## Question
**未成年者** の人権制約（飲酒・喫煙禁止や選挙権制限など）において、成人とは異なる規制が許容される根拠は何か？

## Answer
**パターナリズム（国親思想）**に基づく保護と、健全な育成の必要性。

 **趣旨** : 判断能力が未熟な者を保護し、次代の社会の担い手として育成するため。
 **解説** : 未成年者は精神的・肉体的に発育途上にあり、判断能力が不十分である。そのため、自己決定権に対する干渉（パターナリスティックな制約）や、有害環境からの遮断といった保護的措置が、合理的な区別として広く認められる。
 **条文** : 憲法13条、15条
