# Card: Dockerコンテナやイメージの詳細情報を表示するコマンドは？

## Question
Dockerコンテナやイメージの詳細情報を表示するコマンドは？

## Answer
docker inspect<br><pre><code>docker inspect my-container     # コンテナの詳細情報を表示
docker inspect nginx:latest     # nginxイメージの詳細情報を表示
docker inspect webapp database  # 複数のコンテナ情報を同時表示</code></pre>
