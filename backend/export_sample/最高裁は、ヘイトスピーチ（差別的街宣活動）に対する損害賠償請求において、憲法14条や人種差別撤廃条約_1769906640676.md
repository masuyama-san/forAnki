# Card: 最高裁は、ヘイトスピーチ（差別的街宣活動）に対する損害賠償請求において、憲法14条や人種差別撤廃条約

## Question
最高裁は、ヘイトスピーチ（差別的街宣活動）に対する損害賠償請求において、憲法14条や人種差別撤廃条約をどのように位置づけたか？

## Answer
直接適用の根拠ではなく、 **民法709条の「違法性」判断の基準** として参酌した。

 **趣旨** : 私人間効力における間接適用説の枠組みを維持するため。
 **解説** : 裁判所は「条約や憲法が直接適用される」とは述べず、人種差別的な発言が「社会的に許容しうる限界を超える侮辱行為」であるとして、民法上の不法行為責任を認めた。その違法性判断の背景として憲法等の趣旨を用いた。
 **条文** : 民法709条
 **出典** : 最決平26.12.9（京都朝鮮学校事件）
