# Card: 一元的内在制約説において、「公共の福祉」はどのように定義されるか？

## Question
一元的内在制約説において、「公共の福祉」はどのように定義されるか？

## Answer
人権相互の矛盾・衝突を調整するための実質的公平の原理。

 **趣旨** : 人権は絶対無制約ではなく、他者の人権と共存するために論理必然的な限界を持つことを示すため。
 **解説** : この説は、公共の福祉を「人権の外にある上位概念（国家の利益）」とは考えず、あくまで「人権 vs 人権」の衝突を公平にさばくための内在的な原理とする。したがって、すべての基本的人権はこの原理による制約を当然に受ける。
 **条文** : 憲法12条、13条
