# Card: docker runでコンテナに名前を付けるオプションは？

## Question
docker runでコンテナに名前を付けるオプションは？

## Answer
docker run --name コンテナ名<br><pre><code>docker run --name my-nginx nginx       # my-nginxという名前を付ける
docker run --name db -d mysql          # dbという名前でMySQLを起動
docker run --name web-server -p 80:80 nginx  # 名前とポート設定を組み合わせ</code></pre>
