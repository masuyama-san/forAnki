# Card: 博多駅事件において示された、取材フィルムの提出命令が許容されるための「比較衡量の基準」は？

## Question
博多駅事件において示された、取材フィルムの提出命令が許容されるための「比較衡量の基準」は？

## Answer
「 **公正な裁判の実現** （証拠の不可欠性）」と「 **取材の自由が妨げられる程度** （報道機関の不利益）」を比較衡量して決する。

 **趣旨** : 憲法上の要請である「公正な裁判（37条1項）」と「表現の自由（21条）」という対立利益を調整するため。
 **解説** : 判例は、取材フィルムを証拠とすることが犯罪の立証に不可欠か（代替手段がないか）と、それにより将来の取材活動がどの程度萎縮するかを比較する。博多駅事件では、すでに **放映済み** のフィルムであったため、報道への不利益は限定的であるとして提出命令を合憲とした。
 **条文** : 憲法21条、37条1項、刑事訴訟法99条
 **出典** : 博多駅テレビフィルム提出命令事件（最大決昭44.11.26）
