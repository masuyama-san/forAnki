# Card: 大分県屋外広告物条例事件において、政治的ポスターを街路樹等へ掲出する行為を処罰することは合憲か？

## Question
大分県屋外広告物条例事件において、政治的ポスターを街路樹等へ掲出する行為を処罰することは合憲か？

## Answer
**合憲** 。 **美観風致の維持** は公共の福祉に含まれる正当な目的だから。

 **趣旨** : 「表現の内容」ではなく、表現の「方法（場所）」に対する規制（内容中立規制）であるため。
 **解説** : 最高裁は、都市の美観風致の維持も公共の福祉に含まれるとし、表現の自由（政治活動の自由）といえども、必要かつ合理的な制限を受けるとした。一律禁止ではなく、指定場所以外での掲出を禁じる規制は合理的である。
 **条文** : 憲法21条1項
 **出典** : 最判昭62.3.3（大分県屋外広告物条例事件）
