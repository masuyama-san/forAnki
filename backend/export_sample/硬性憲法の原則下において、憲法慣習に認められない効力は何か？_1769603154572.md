# Card: 硬性憲法の原則下において、憲法慣習に認められない効力は何か？

## Question
硬性憲法の原則下において、憲法慣習に認められない効力は何か？

## Answer
憲法典の明文規定を改廃する効力。

 **趣旨** : 改正手続（96条）を経ない実質的な憲法変更（憲法変遷）が安易に認められれば、硬性憲法の趣旨が没却される。 
 **条文** : 第96条、第98条1項。 
 **判例・あてはめ** : 憲法慣習は「補充的」な効力に留まり、明白な明文違反を正当化する「改廃的」な効力は否定されるのが通説的見解である。
