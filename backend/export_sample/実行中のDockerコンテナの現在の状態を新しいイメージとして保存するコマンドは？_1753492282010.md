# Card: 実行中のDockerコンテナの現在の状態を新しいイメージとして保存するコマンドは？

## Question
実行中のDockerコンテナの現在の状態を新しいイメージとして保存するコマンドは？

## Answer
docker commit<br>（一時的な用途やデバッグ用途として使うのが一般的で、本番環境のイメージ構築には <code>Dockerfile</code> を使うべき）<br><pre><code>docker commit my-container my-new-image        # コンテナをイメージとして保存
docker commit webapp myapp:v1.1               # タグ付きで保存
docker commit c1d2e3f4 custom-ubuntu          # コンテナIDでイメージ作成</code></pre>
