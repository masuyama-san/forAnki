# Card: 小売市場距離制限事件（最大判昭47.11.22）において、距離制限が「合憲」とされた理由（ロジック）

## Question
小売市場距離制限事件（最大判昭47.11.22）において、距離制限が「合憲」とされた理由（ロジック）は？

## Answer
中小小売商の保護という「積極目的」のための措置であり、著しく不合理であることが明白とはいえないから。

 **趣旨** : 過当競争による共倒れ防止という政策判断は、立法府の裁量事項であるため。
 **解説** : 本判決は、規制目的を「中小企業保護（積極目的）」と認定した。その上で、距離制限という手段が政策として著しく不合理とはいえないため、明白性の基準により合憲とした。薬局判決（消極目的・違憲）との対比が最重要。
 **条文** : 憲法第22条第1項
 **出典** : 最大判昭47.11.22
