# Card: 憲法の人権規定は、私人間の行為に対して原則として適用されるか？

## Question
憲法の人権規定は、私人間の行為に対して原則として適用されるか？

## Answer
**適用されない** 。 
（間接適用説を採用）

 **趣旨** : 「私的自治の原則」を尊重し、国家権力が市民の私生活に過干渉となるのを防ぐため。
 **解説** : 憲法は本来、公権力による人権侵害から国民を守る「対国家規範」である。私人相互の関係には「私的自治」が妥当するため、原則として憲法は適用されない。ただし、社会的許容限度を超える侵害については、民法の一般条項を通じて間接的に規律する。
 **条文** : 民法1条（私権の基本原則）
 **出典** : 最大判昭48.12.12（三菱樹脂事件）
