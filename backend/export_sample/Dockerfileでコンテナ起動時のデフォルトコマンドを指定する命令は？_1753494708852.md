# Card: Dockerfileでコンテナ起動時のデフォルトコマンドを指定する命令は？

## Question
Dockerfileでコンテナ起動時のデフォルトコマンドを指定する命令は？

## Answer
CMD<br><pre><code>CMD ["python", "app.py"]           # Pythonアプリケーションを起動
CMD ["nginx", "-g", "daemon off;"] # nginxをフォアグラウンドで起動
CMD ["npm", "start"]               # Node.jsアプリケーションを起動</code></pre>
