# Card: docker pullですべてのタグのイメージを一度にダウンロードするオプションは？

## Question
docker pullですべてのタグのイメージを一度にダウンロードするオプションは？

## Answer
docker pull -a イメージ名<br>-a："all"（すべて）<br><pre><code>docker pull -a ubuntu          # ubuntuのすべてのタグをダウンロード
docker pull --all-tags python  # pythonのすべてのバージョンをダウンロード</code></pre>
