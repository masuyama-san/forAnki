# Card: 日産自動車事件において、女性の定年年齢を男性より低く定めた就業規則の規定は有効とされたか？

## Question
日産自動車事件において、女性の定年年齢を男性より低く定めた就業規則の規定は有効とされたか？

## Answer
**無効** （公序良俗違反）。

 **趣旨** : 企業内の優越的地位の濫用を是正し、労働者の平等を実質的に守るため。
 **解説** : 企業の経営上の裁量を考慮しても、専ら女子であることを理由とする差別的取り扱いは、合理的な理由がなく、憲法14条の趣旨に反し、民法90条の公序良俗に違反して無効である。
 **条文** : 民法90条、憲法14条
 **出典** : 最判昭56.3.24（日産自動車事件）
