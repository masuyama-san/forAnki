# Card: 民間の分譲マンション共用部分への「政党ビラ」配布立ち入り（葛飾マンション事件）について、住居侵入罪の

## Question
民間の分譲マンション共用部分への「政党ビラ」配布立ち入り（葛飾マンション事件）について、住居侵入罪の成立を認めることは合憲か？

## Answer
**合憲** 。
 **政治的ビラ** であっても商業ビラと同様、管理者の意思に反する立ち入りは **私生活の平穏** を害するから。

 **趣旨** : 私人の管理する場所においては、居住者の「私生活の平穏」と管理権が優先される。
 **解説** : 判例は、マンション共用部分は住人の私的生活の場であり、管理組合の意思に反する立ち入りは「私生活の平穏」を害するとした。表現の自由の重要性を考慮しても、刑罰法令の適用は違憲ではない。
 **条文** : 憲法21条1項、刑法130条
 **出典** : 最判平21.11.30
