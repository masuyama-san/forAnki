# Card: docker tagでレジストリ用のタグを付ける方法は？

## Question
docker tagでレジストリ用のタグを付ける方法は？

## Answer
docker tag ソースイメージ レジストリURL/イメージ名:タグ<br><pre><code>docker tag my-app:latest registry.company.com/my-app:v1.0     # 企業レジストリ用
docker tag webapp:latest your-account.dkr.ecr.region.amazonaws.com/webapp:prod  # AWS ECR用
docker tag my-image gcr.io/project-id/my-image:latest         # Google Container Registry用</code></pre>
