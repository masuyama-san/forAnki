# Card: Dockerイメージをレジストリにアップロードするコマンドは？

## Question
Dockerイメージをレジストリにアップロードするコマンドは？

## Answer
docker push<br><pre><code>docker push my-image:latest        # Docker Hubにイメージをプッシュ
docker push username/app:v1.0      # バージョン指定でプッシュ
docker push registry.company.com/my-app:prod  # プライベートレジストリにプッシュ</code></pre>
