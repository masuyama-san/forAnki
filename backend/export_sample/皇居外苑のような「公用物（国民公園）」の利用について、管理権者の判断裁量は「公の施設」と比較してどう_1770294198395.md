# Card: 皇居外苑のような「公用物（国民公園）」の利用について、管理権者の判断裁量は「公の施設」と比較してどう

## Question
皇居外苑のような「公用物（国民公園）」の利用について、管理権者の判断裁量は「公の施設」と比較してどう扱われるか？

## Answer
「公の施設」よりも **広範な裁量** が認められる。

（ただし、管理権者の単なる自由裁量ではなく、著しく妥当性を欠く場合は違法となる）

 **趣旨** : 公園等は集会専用の施設ではなく、一般公衆の休息や環境保全（本来の用途）との調整が必要となるため。
 **解説** : 市民会館等の「公の施設」は利用させること自体が目的であるため原則利用権があるが、「公用物」はそうではない。したがって、管理権者が本来の用途との兼ね合いで不許可とする裁量の幅は広いが、絶対的なものではない。
 **条文** : 憲法21条1項
 **出典** : 最大判昭28.12.23（皇居前広場事件）
