# Card: 私立高校の校則（パーマ禁止等）による生徒の退学処分について、裁判所が有効性を判断する基準は？

## Question
私立高校の校則（パーマ禁止等）による生徒の退学処分について、裁判所が有効性を判断する基準は？

## Answer
校則の内容が **社会通念上不合理** とはいえず、処分が**社会観念上著しく妥当を欠く（裁量権の逸脱・濫用）**ものでない限り有効。

 **趣旨** : 私的自治（私立学校の教育の自由）と、学生の一般的行動の自由の調整。
 **解説** : 憲法13条は直接適用されず、民法1条・90条等の解釈を通じて間接適用される。高校生らしい規律を求める校則は不合理ではなく、是正に従わず違反を繰り返した生徒への退学処分も裁量の範囲内とした。
 **条文** : 民法1条・90条
 **出典** : 最判平8.7.18（修徳学園パーマ事件）
