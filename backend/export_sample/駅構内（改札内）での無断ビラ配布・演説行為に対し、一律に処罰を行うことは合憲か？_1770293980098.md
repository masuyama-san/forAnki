# Card: 駅構内（改札内）での無断ビラ配布・演説行為に対し、一律に処罰を行うことは合憲か？

## Question
駅構内（改札内）での無断ビラ配布・演説行為に対し、一律に処罰を行うことは合憲か？

## Answer
**合憲** 。駅はパブリック・フォーラムではなく、管理者の許諾のない行為は **管理権を不当に害する** から。

 **趣旨** : 駅の公共性はあるものの、鉄道事業者の管理権や利用者の安全・円滑な利用を保護する必要があるため。
 **解説** : 駅構内は「パブリック・フォーラム（公園や道路など、表現の場として開放された公共空間）」ではなく、管理者が旅客の安全等のために管理権を行使する場所である。管理者の許諾を受けずに行われた演説等は、管理権を不当に害するものであり、処罰も許される。
 **条文** : 憲法21条1項
 **出典** : 最判昭59.12.18（駅構内ビラ配り事件）
