# Card: 司法書士会 （強制加入団体）が、会員から 震災復興支援 のための負担金を強制徴収することは許されるか

## Question
**司法書士会** （強制加入団体）が、会員から **震災復興支援** のための負担金を強制徴収することは許されるか？

## Answer
許される（有効）。

 **趣旨** : 政治的・宗教的立場に関わらない「公益活動」への協力は、個人の思想信条の核心を害さないため。
 **解説** : 阪神・淡路大震災の復興支援拠出金のための徴収は、会員の政治的・思想的立場を害するものではなく、司法書士会の目的の範囲内として有効とされた。南九州税理士会事件（政治献金＝無効）との決定的違いは「活動の政治性の有無」である。
 **条文** : 民法34条
