# Card: 集団行動（デモ行進）について、条例で「許可制」を設けることは憲法21条に違反するか？

## Question
集団行動（デモ行進）について、条例で「許可制」を設けることは憲法21条に違反するか？

## Answer
違反しない（合憲）。

ただし、許可の条件が **合理的かつ明確** であり、不許可事由が厳格に限定されている必要がある。

 **趣旨** : 集団行動は「物理的エネルギー」を伴い、一般交通や平穏な静穏と衝突する可能性があるため、事前の調整（時・所・方法の規制）が必要だから。
 **解説** : 通常の表現行為への許可制は原則違憲だが、デモ行進の調整に限る許可制は、「検閲」には当たらず許容される。判例は、実質的な「届出制」に近い運用が担保されている限り合憲とする。
 **条文** : 憲法21条1項
 **出典** : 最大判昭29.11.24（新潟県公安条例事件）
