# Card: 公務員批判などに対する「事前差止め」が、例外的に許容されるための要件（北方ジャーナル基準）は？

## Question
公務員批判などに対する「事前差止め」が、例外的に許容されるための要件（北方ジャーナル基準）は？

## Answer
- その表現内容が **真実でないことが明白** であること。
- 被害者が **重大にして著しく回復困難な損害** を被るおそれがあること。

 **趣旨** : 事前差止めは「検閲」に類似する強力な規制（事前抑制）であるため、原則禁止とし、例外を極めて厳格に限定するため。
 **解説** : 名誉毀損に対する事後的な損害賠償とは異なり、「明白性（一見して嘘とわかる）」と「重大性（人格権の回復不能な侵害）」という加重された要件が必要。このハードルは極めて高く、実質的にはほとんど認められない。
 **条文** : 憲法21条2項（検閲禁止の趣旨）
 **出典** : 北方ジャーナル事件（最大判昭61.6.11）
