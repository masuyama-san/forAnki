# Card: 天皇・皇族 に基本的人権は保障されるか？

## Question
**天皇・皇族** に基本的人権は保障されるか？

## Answer
「国民」に含まれ原則保障されるが、 **最小限の特例的な制約** を受ける。

 **趣旨** : 天皇の地位は「主権者の総意」に基づく公的なものであり、憲法上の制度的要請（象徴・世襲）が優先されるため。
 **解説** : 天皇も人間としての尊厳を有するため人権享有主体性は認められる（通説）。しかし、憲法第1章の規定により、選挙権の欠如、政治活動の禁止、婚姻の自由の制約（皇室会議の議決）など、地位に伴う内在的な制約を受ける。
 **条文** : 憲法1条、2条、第3章
