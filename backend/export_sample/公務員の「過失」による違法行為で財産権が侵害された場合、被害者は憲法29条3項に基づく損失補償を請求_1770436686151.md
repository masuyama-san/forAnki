# Card: 公務員の「過失」による違法行為で財産権が侵害された場合、被害者は憲法29条3項に基づく損失補償を請求

## Question
公務員の「過失」による違法行為で財産権が侵害された場合、被害者は憲法29条3項に基づく損失補償を請求できるか？

## Answer
できない（国家賠償法に基づく損害賠償を請求すべき）

 **趣旨** : 損失補償は「適法行為」に伴う損失の公平負担制度であり、違法行為に対するペナルティや原状回復としての損害賠償とは法的性質が異なるからである。
 **解説** : 違法な公権力の行使によって損害が生じた場合は、憲法17条および国家賠償法に基づく損害賠償請求の問題となる。憲法29条3項はあくまで「適法」な公権力行使（公共のために用ひる）場合を想定しているため、違法行為に対して損失補償を請求することはできない。
 **条文** : 憲法17条、29条3項
