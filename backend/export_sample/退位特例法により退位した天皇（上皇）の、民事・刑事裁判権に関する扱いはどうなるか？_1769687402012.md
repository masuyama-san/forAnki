# Card: 退位特例法により退位した天皇（上皇）の、民事・刑事裁判権に関する扱いはどうなるか？

## Question
退位特例法により退位した天皇（上皇）の、民事・刑事裁判権に関する扱いはどうなるか？

## Answer
天皇の例による（原則として裁判権は及ばない）。

 **趣旨** : 退位後も、元天皇としての尊厳を維持し、在位中の行為に対する追及による政治的混乱を防ぐため。 
 **解説** : 「天皇の退位等に関する皇室典範特例法」により、上皇の身分に関する事項は原則として天皇の例による。特例法附則4条1項1号では、上皇に関しては刑事裁判権や検察審査会の職務などについて「天皇の例による」と規定されており、不訴追特権が維持されている。また、記帳所事件の論理を援用すれば、象徴であった地位の特殊性から、民事裁判権についても制限されると解される。 
 **条文** : 天皇の退位等に関する皇室典範特例法3条、附則4条
