# Card: 破産手続開始の決定を受けた者（破産者）に対する居住の自由の制約（裁判所の許可なく居住地を離れられない

## Question
破産手続開始の決定を受けた者（破産者）に対する居住の自由の制約（裁判所の許可なく居住地を離れられない）の根拠は何か？

## Answer
公正な清算・債権者保護のための「経済的自由」の制約。

 **趣旨** : 破産者の逃亡や財産隠匿を防ぎ、破産手続き（説明義務等）を円滑に進めるため。
 **解説** : 居住・移転の自由における「経済的自由」の側面の典型的な制約例。この制約は手続き期間中に限定されるものであり、公共の福祉による合理的な制約（政策的制約）とされる。目的の正当性と手段の合理性が肯定されやすい類型の規制である。
 **条文** : 憲法22条1項、破産法37条1項
