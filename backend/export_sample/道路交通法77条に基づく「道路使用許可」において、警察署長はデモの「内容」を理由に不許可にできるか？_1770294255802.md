# Card: 道路交通法77条に基づく「道路使用許可」において、警察署長はデモの「内容」を理由に不許可にできるか？

## Question
道路交通法77条に基づく「道路使用許可」において、警察署長はデモの「内容」を理由に不許可にできるか？

## Answer
できない。

警察署長の権限は、 **交通の安全と円滑** を図るための技術的・客観的な調整に限られる。

 **趣旨** : 警察署長に内容審査権を与えれば、実質的な検閲となり、表現の自由を侵害するから。
 **解説** : 道路交通法は交通警察の法規であり、思想警察の法規ではない。不許可にできるのは「一般交通に著しい支障を及ぼし、条件を付しても回避できない場合」に限られる。集会の目的やスローガンを理由とした不許可は、法の委任の範囲を逸脱し違法となる。
 **条文** : 憲法21条、道路交通法77条
 **出典** : 最判昭57.11.16
