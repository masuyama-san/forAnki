# Card: 私人間効力の問題において、人権侵害行為が「事実行為（差別発言・暴力・嫌がらせ）」である場合、救済のた

## Question
私人間効力の問題において、人権侵害行為が「事実行為（差別発言・暴力・嫌がらせ）」である場合、救済のために用いる民法の条文は？

## Answer
民法709条（不法行為）

 **趣旨** : 「無効」という概念が観念できない事実行為に対し、金銭賠償による事後的救済を図るため。
 **解説** : 差別的発言などの事実行為については、憲法の趣旨を参酌して被侵害利益の重要性を評価し、受忍限度を超える場合に「違法性」を認めて損害賠償を命じる。
 **条文** : 民法709条
 **出典** : 最決平26.12.9（京都朝鮮学校事件参照）
