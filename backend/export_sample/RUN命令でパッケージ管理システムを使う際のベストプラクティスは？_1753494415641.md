# Card: RUN命令でパッケージ管理システムを使う際のベストプラクティスは？

## Question
RUN命令でパッケージ管理システムを使う際のベストプラクティスは？

## Answer
update + install + clean を1つのRUNで実行<br><pre><code># Ubuntu/Debian
RUN apt-get update &amp;&amp; \
    apt-get install -y python3 curl &amp;&amp; \
    apt-get clean &amp;&amp; \
    rm -rf /var/lib/apt/lists/*

# Alpine Linux
RUN apk add --no-cache python3 curl

# CentOS/RHEL
RUN yum update -y &amp;&amp; \
    yum install -y python3 curl &amp;&amp; \
    yum clean all</code></pre>
