# Card: 「審級制度（三審制など）」は、憲法上の権利として保障されているか？（例：最高裁を頂点とする二審制にす

## Question
「審級制度（三審制など）」は、憲法上の権利として保障されているか？（例：最高裁を頂点とする二審制にすることは合憲か？）

## Answer
保障されていない。
合憲である。

 **趣旨** : 裁判の適正さと迅速さのバランスをどうとるかは、その時代の要請や事件の特性に応じた立法政策に委ねるべきだからである。
 **解説** : 憲法は「最高裁判所」が終審裁判所であることを規定している（76条、81条）以外、具体的な審級の数については沈黙している。したがって、三審制は憲法上の要請ではなく、特定の事件について二審制としたり上告を制限したりすることも、立法府の裁量の範囲内であり合憲である。
 **条文** : 憲法32条、76条、81条
 **出典** : 最大判昭23.3.10
